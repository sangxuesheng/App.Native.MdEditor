# 更新日志 (Changelog)

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.27.6] - 2026-03-08

### 新增
- 完善微信公众号导出功能
- Mermaid 流程图自动转换为 SVG Data URL 图片
- 数学公式使用 CodeCogs API 在线渲染为高清 PNG（300 DPI）

### 改进
- 从 KaTeX annotation 标签提取原始 LaTeX 代码
- 图片缩放：百分比宽度自动转换为像素值（基于微信公众号 677px 标准容器）
- 优化导出流程，支持异步处理
- 添加详细的调试日志

### 修复
- 修复图片样式处理逻辑
- 修复数学公式在微信编辑器中显示不清晰的问题

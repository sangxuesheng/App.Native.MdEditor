# 更新日志 (Changelog)

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.28.7] - 2026-03-15

### 新增
- AI 文生图尺寸选项增加比例图标（正方形/横版/竖版），直观展示各尺寸样式
- 参考图、图片数量、种子三处增加灰色提示文案，说明支持范围与用法

### 改进
- 文生图 endpoint 选择逻辑修复：选择非 builtin 服务商（硅基流动、OpenAI 等）时不再误用内置代理
- Fal Kolors 支持多图生成，传递 `num_images` 参数（1–8 张）

### 修复
- 修复切换文生图服务商时 endpoint 未同步更新导致仍走 proxy-ai.doocs.org 的问题

---

## [1.28.6] - 2026-03-15

### 新增
- AI 配置：模型标签页（对话/图片）移至顶部，API Key 共用，代理地址按 tab 区分
- 图片 tab 支持独立代理地址配置（imageConfig.endpoints），默认使用对话代理地址
- 连通性检查随 tab 切换：对话 tab 测对话 API，图片 tab 测图片 API，模型选项自动切换
- 模型列表行内连通性检查：结果以浮窗显示在试管按钮旁，不占用上方通知区

### 改进
- 保存/重置按钮固定在右侧面板底部，左侧服务列表贯穿全高
- 点击模型「使用」开关后立即持久化到数据库，无需再点保存
- 连通性检查通过后不再自动启用模型，由用户自行在列表中点击使用
- 主标签按钮布局参考底部操作区，未选中状态也显示边框
- 图片模式下，服务商无图片模型时显示「服务商暂无图片相关模型」

## [1.28.5] - 2026-03-15

### 新增
- AI 对话「引用全文」启用时，输入区显示引用框，与消息中的引用块风格一致

### 改进
- AI 对话引用功能改为只同时引用一个：快捷指令或 AI 回复引用会替换已有引用，启用引用全文会清空其他引用块
- 引用全文块仅显示「引用全文」四字，悬停显示内容预览
- getEditorContent 回退逻辑优化，确保在编辑器未挂载时仍能正确获取 content 状态

## [1.28.2] - 2026-03-15

### 新增
- 安装向导支持用户自定义应用端口，安装前自动检查端口占用

### 改进
- 应用入口改为 iframe 模式，在飞牛桌面内打开
- 发布按钮从菜单栏移至工具栏右侧（导出配置与保存之间）
- 非飞牛桌面下点击发布直接打开 SyncDialog，由弹窗内部检测扩展并显示（检测中/未安装引导/平台选择），不再使用 toast 提示

### 修复
- manifest 设置 checkport=false，端口检查由 install_init 负责，支持用户自定义端口

## [1.28.1] - 2026-03-14

### 新增
- AI 对话快捷指令「帮我写主题」，根据自定义 CSS 主题规范生成主题代码
- 快捷指令改为引用块插入，提示词不写入编辑区，便于查看与补充

### 改进
- 文生图生成后自动保存到图片库（不压缩），插入时使用本地路径
- 插入图片/链接到文档时，上下各留一行空格
- 链接按钮改为插入 `[链接](https://)` 模板
- 上传图片、粘贴、拖拽插入时，上下各留一行空格

### 修复
- 修复 API 返回 500 或空响应时的 `Unexpected end of JSON input` 崩溃，新增 `safeParseJsonResponse` 安全解析
- 修复 FileTree `onSelectStart` 非标准事件导致的 React 警告，改用 CSS `user-select: none`

## 常见问题排查

### 问题 1: 图注不在图片下方
**可能原因：**
- 图片不是 `<figure>` 元素
- CSS 没有正确加载

**检查方法：**
1. 查看 HTML 源代码
2. 确认图片结构：
   ```html
   <figure class="image-figure">
     <img src="..." alt="..." title="...">
     <figcaption>标题</figcaption>
   </figure>
   ```

### 问题 2: 代码行号不显示
**可能原因：**
- 没有启用"代码块行号"选项
- `data-line-numbers` 属性缺失
- CSS 没有正确加载

**检查方法：**
1. 确认已启用"代码块行号"
2. 查看 HTML 源代码
3. 确认代码块结构：
   ```html
   <pre><code class="line-numbers" data-line-numbers="1
2
3
4">...</code></pre>
   ```

### 问题 3: 样式不生效
**可能原因：**
- CSS 优先级问题
- 浏览器缓存

**解决方案：**
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 强制刷新（Ctrl+F5）
3. 检查 `<style>` 标签是否在 `<head>` 中

## 📈 技术栈

### 前端
- React 18
- Monaco Editor
- Vite 5
- markdown-it
- markdown-it-task-lists
- markdown-it-footnote
- markdown-it-katex (数学公式)
- markdown-it-mark (高亮文本)
- KaTeX (本地打包)
- Mermaid (CDN)
- Lucide React (图标)

### 后端
- Node.js (CommonJS)
- HTTP Server
- 文件系统 API
- 图片上传处理

### 构建
- Vite (ES2020 目标)
- esbuild (压缩)
- 代码分割（React/Markdown vendor）
- CSS 代码分割

## ✨ 实现的功能

### 前端功能
✅ **图片管理对话框**
- 4个标签页：上传图片、图片链接、图片库、图床设置
- 拖拽上传支持
- 点击选择文件上传
- 多文件上传
- 上传进度提示
- 最近上传图片展示
- 图片插入到编辑器

✅ **图片链接插入**
- 手动输入图片 URL
- 可选图片描述
- 一键插入 Markdown 语法

✅ **主题适配**
- 浅色主题 (light)
- 深色主题 (dark)
- MD3 主题

✅ **占位功能**
- 图片库（显示空状态）
- 图床设置（显示开发中提示）

### 后端功能
✅ **图片上传 API** (`POST /api/image/upload`)
- multipart/form-data 解析
- 文件类型验证（JPG/PNG/GIF/WebP）
- 文件大小限制（10MB）
- 按日期组织存储 (`/images/YYYY/MM/DD/`)
- 唯一文件名生成
- 返回图片 URL 和元数据

✅ **图片静态服务** (`GET /images/*`)
- 正确的 MIME 类型
- 缓存控制（1年）
- 路径安全验证

## 📝 使用示例

### 用户操作流程
```
1. 点击工具栏"图片"按钮
   ↓
2. 选择"上传图片"标签页
   ↓
3. 拖拽图片到上传区域
   ↓
4. 等待上传完成
   ↓
5. 点击图片上的"插入"按钮
   ↓
6. 图片自动插入到编辑器
   ↓
7. 预览区域显示图片
```

### 生成的 Markdown
```markdown
![photo](/images/2026/02/27/1709049600000_abc123.jpg)
```

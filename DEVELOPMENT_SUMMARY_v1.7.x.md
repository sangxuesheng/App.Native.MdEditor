## 🐛 遇到的问题

### 1. MenuBar.css 文件清空问题
**问题**: 使用 StrReplace 工具导致文件被清空  
**原因**: 工具匹配失败时会清空文件  
**解决**: 改用 Python 脚本或 sed 命令  
**教训**: 修改前先备份，使用更安全的修改方法

### 2. 函数未定义错误
**问题**: MenuBar 调用未定义的处理函数  
**原因**: Python 脚本没有成功插入函数  
**解决**: 手动添加缺失的函数  
**教训**: 修改后验证关键代码是否存在

### 3. Git restore 失效
**问题**: git restore 没有正确恢复文件  
**原因**: 文件已被修改但未提交  
**解决**: 从 git 历史中提取正确版本  
**教训**: 重要修改前先提交

## 🔧 技术实现

### 前端上传流程

```javascript
1. 用户选择/拖拽图片
   ↓
2. 验证文件类型和大小
   ↓
3. 创建 FormData
   ↓
4. POST /api/image/upload
   ↓
5. 接收返回的 URL
   ↓
6. 显示在"最近上传"
   ↓
7. 用户点击"插入"
   ↓
8. 插入 Markdown 语法到编辑器
```

### 后端存储结构

```
授权目录/
└── images/
    └── 2026/
        └── 02/
            └── 27/
                ├── 1709049600000_abc123.jpg
                ├── 1709049601000_def456.png
                └── ...
```

### API 请求示例

#### 上传图片
```http
POST /api/image/upload
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary

------WebKitFormBoundary
Content-Disposition: form-data; name="images"; filename="photo.jpg"
Content-Type: image/jpeg

[二进制数据]
------WebKitFormBoundary--
```

#### 响应
```json
{
  "ok": true,
  "images": [
    {
      "url": "/images/2026/02/27/1709049600000_abc123.jpg",
      "filename": "photo.jpg",
      "size": 102400,
      "alt": "photo"
    }
  ]
}
```

## 📝 使用说明

### 用户操作流程

1. **打开图片管理器**
   - 点击工具栏的"图片"按钮
   - 或使用菜单：插入 → 图片

2. **上传图片**
   - 方式1：拖拽图片到上传区域
   - 方式2：点击"选择图片"按钮
   - 支持多文件上传

3. **插入图片**
   - 上传成功后，图片显示在"最近上传"
   - 点击图片上的"插入"按钮
   - 自动插入 Markdown 语法：`![alt](url)`

4. **使用图片链接**
   - 切换到"图片链接"标签页
   - 输入图片 URL
   - 可选：输入图片描述
   - 点击"插入图片"

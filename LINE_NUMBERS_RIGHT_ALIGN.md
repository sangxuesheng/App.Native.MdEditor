# ✅ 行号右对齐

## 完成时间

## 修改内容

### 行号对齐方式调整
将行号从左对齐改为右对齐，使数字更接近折叠符号。

## CSS 实现

### 添加的样式

```css
/* ==================== 行号右对齐 ==================== */
.monaco-editor .line-numbers {
  text-align: right !important;
  padding-right: 8px !important;
}
```

## 样式说明

### text-align: right
- 将行号数字右对齐
- 数字靠近折叠符号
- 视觉上更紧凑

### padding-right: 8px
- 在行号和折叠符号之间保留 8px 间距
- 避免数字和符号贴得太近
- 保持可读性

## 设计理念

### 1. 视觉紧凑
- 行号和折叠符号形成视觉组
- 减少不必要的空白
- 界面更整洁

### 2. 对齐一致
- 所有折叠符号在同一垂直线上
- 不受行号位数影响
- 视觉上更统一

### 3. 符合习惯
- VS Code 使用右对齐行号
- 大多数现代编辑器都是右对齐
- 用户更熟悉这种布局

## 间距调整

### padding-right: 8px

**为什么是 8px？**

**计算**:
```
行号区域宽度: 50px
行号数字宽度: 约 30px (3个字符)
右边距: 8px
折叠图标位置: 30px
折叠图标宽度: 16px
```

**视觉效果**:
```
│  1  ⌄│
   ↑↑ ↑
   数字 间距 折叠图标
   右对齐 8px
```

### 如果需要调整间距

**更近一些**:
```css
.monaco-editor .line-numbers {
  padding-right: 4px !important;  /* 更近 */
}
```

**更远一些**:
```css
.monaco-editor .line-numbers {
  padding-right: 12px !important;  /* 更远 */
}
```

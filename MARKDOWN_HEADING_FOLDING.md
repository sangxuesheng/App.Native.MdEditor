# ✅ Markdown 标题折叠功能

## 完成时间
2026-02-27 01:15

## 功能说明
启用 Markdown 标题的折叠功能，可以折叠一级标题、二级标题及其下的所有内容。

## 修改内容

### 折叠策略调整
从 `'indentation'`（基于缩进）改为 `'auto'`（自动检测）

```javascript
// 之前
foldingStrategy: 'indentation',  // 基于缩进，不适合 Markdown 标题

// 之后
foldingStrategy: 'auto',         // 自动检测，支持 Markdown 标题
```

## 折叠规则

### 1. 一级标题 (#)
- 折叠范围：从当前行到下一个一级标题之前
- 包含内容：所有二级、三级标题及其内容

### 2. 二级标题 (##)
- 折叠范围：从当前行到下一个同级或更高级标题之前
- 包含内容：所有三级、四级标题及其内容

### 3. 三级标题 (###)
- 折叠范围：从当前行到下一个同级或更高级标题之前
- 包含内容：所有四级、五级标题及其内容

### 4. 以此类推
- 支持最多 6 级标题（Markdown 标准）
- 每级标题都可以独立折叠

## 使用场景

### 1. 长文档大纲浏览
```markdown

▶ # 第二章 ...
▶ # 第三章 ...
▶ # 第四章 ...
```
折叠所有章节，快速浏览文档结构

### 2. 专注编辑某一章节
```markdown

▼ # 第二章
  ## 2.1 小节
  正在编辑的内容...
  ## 2.2 小节
▶ # 第三章 ...
```
折叠其他章节，专注当前编辑

### 3. 分层查看内容
```markdown
▼ # 第一章
  ▶ ## 1.1 小节 ...
  ▼ ## 1.2 小节
    ### 1.2.1 详细内容
    ### 1.2.2 更多内容
  ▶ ## 1.3 小节 ...
```
选择性展开需要查看的部分

## 配置说明

### foldingStrategy: 'auto'

**工作原理**:
- 自动检测文件类型（Markdown）
- 使用 Markdown 语法规则识别可折叠区域
- 基于标题层级确定折叠范围

**优势**:
- ✅ 支持 Markdown 标题折叠
- ✅ 智能识别标题层级
- ✅ 符合 Markdown 文档结构
- ✅ 与 VS Code 行为一致

**对比 'indentation'**:
- `'indentation'`: 基于缩进，不适合 Markdown（标题无缩进）
- `'auto'`: 基于语法，完美支持 Markdown 标题

## 快捷键

### 折叠操作
- `Ctrl/Cmd + Shift + [` - 折叠当前区域
- `Ctrl/Cmd + K, Ctrl/Cmd + 0` - 折叠所有区域
- `Ctrl/Cmd + K, Ctrl/Cmd + 1` - 折叠所有一级标题
- `Ctrl/Cmd + K, Ctrl/Cmd + 2` - 折叠所有二级标题

### 展开操作
- `Ctrl/Cmd + Shift + ]` - 展开当前区域
- `Ctrl/Cmd + K, Ctrl/Cmd + J` - 展开所有区域

### 切换操作
- `Ctrl/Cmd + K, Ctrl/Cmd + L` - 切换当前折叠状态

# Markdown 编辑器 v1.5.0 快速开始

## 🚀 快速安装

### 方法1: 命令行安装
```bash
appcenter-cli install-local App.Native.MdEditor2.fpk
```

### 方法2: Web UI 安装
1. 打开飞牛 NAS 应用中心
2. 点击"本地安装"
3. 上传 `App.Native.MdEditor2.fpk`
4. 按照向导完成安装

## 📖 使用指南

### 启动应用
- 在应用中心找到"Markdown 编辑器 v2"
- 点击启动
- 或者右键 .md 文件选择"打开方式" → "Markdown 编辑器"

## 🎯 核心功能

### 1. 菜单栏（新增）
**文件菜单**
- 新建文件 (Ctrl+N)
- 保存 (Ctrl+S)
- 另存为 (Ctrl+Shift+S)
- 导出 → 多种格式

**编辑菜单**
- 撤销/重做 (Ctrl+Z/Y)
- 复制/粘贴 (Ctrl+C/V)
- 查找/替换 (Ctrl+F/H)
- 格式化文档 (Shift+Alt+F)

**格式菜单**
- 标题 → 一至六级 (Ctrl+1-6)
- 加粗/斜体 (Ctrl+B/I)
- 列表、引用、代码

**插入菜单**
- 链接 (Ctrl+K)
- 图片、表格
- 代码块、公式、图表

### 2. 文件操作
- **新建**: 支持6种模板（空白、笔记、博客等）
- **保存**: 自动保存（可配置间隔）
- **另存为**: 保存到新位置
- **导出**: HTML、Markdown、TXT 格式

### 3. 编辑功能
- **实时预览**: 即时渲染 Markdown
- **语法高亮**: Monaco Editor
- **快捷键**: 完整的快捷键支持
- **工具栏**: 快速插入常用格式

### 4. 高级功能
- **LaTeX 公式**: 支持数学公式
- **Mermaid 图表**: 流程图、时序图等
- **任务列表**: `- [ ]` 和 `- [x]`
- **表格**: 完整的表格支持

## 🎨 主题切换

### 切换方法
1. 点击菜单栏 → 样式 → 切换主题
2. 点击工具栏右侧的 🌙/☀️ 图标
3. 使用快捷键 `Ctrl+T`

### 主题特点
- **深色主题**: 专业的深色配色，适合长时间编辑
- **浅色主题**: 清爽的浅色配色，适合白天使用

## 📁 文件管理

### 文件树
- 左侧显示文件树
- 点击文件打开
- 支持搜索和过滤
- 点击"文件"按钮切换显示

### 新建文件
1. 点击菜单栏 → 文件 → 新建
2. 选择模板（空白、笔记、博客等）
3. 选择保存位置
4. 输入文件名
5. 点击"创建"

### 自动保存
- 默认每30秒自动保存
- 可在设置中调整间隔
- 工具栏显示"自动"表示已启用
- 点击切换为"手动"模式

## 📤 导出功能

### 导出格式
1. **公众号格式**: 适合微信公众号
2. **HTML 格式**: 完整样式
3. **HTML 格式（无样式）**: 纯 HTML
4. **HTML 格式（兼容样式）**: 兼容性样式
5. **MD 格式**: Markdown 源文件
6. **复制选中内容**: 复制到剪贴板

### 导出步骤
1. 点击菜单栏 → 文件 → 导出
2. 选择导出格式
3. 点击"导出"按钮
4. 文件自动下载

## 🔍 Markdown 语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式
```markdown
**加粗文本**
*斜体文本*
~~删除线~~
`行内代码`
```

### 列表
```markdown
- 无序列表项
- 无序列表项

1. 有序列表项
2. 有序列表项

- [ ] 任务列表
- [x] 已完成任务
```

### 链接和图片
```markdown
[链接文本](https://example.com)
![图片描述](https://example.com/image.jpg)
```

### 代码块
````markdown
```javascript
function hello() {
  console.log('Hello World')
}
```
````

### 表格
```markdown
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
```

### 数学公式
```markdown
行内公式：$E = mc^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### Mermaid 图表
````markdown
```mermaid
graph LR
    A[开始] --> B[处理]
    B --> C[结束]
```
````

## 🐛 常见问题

### Q: 如何打开 .md 文件？
A: 右键 .md 文件 → 打开方式 → Markdown 编辑器

### Q: 自动保存在哪里？
A: 自动保存到原文件位置，草稿保存在浏览器本地存储

### Q: 如何导出 PDF？
A: 导出 HTML 后，使用浏览器的打印功能保存为 PDF

### Q: 支持哪些 Markdown 语法？
A: 支持 GFM（GitHub Flavored Markdown）、LaTeX、Mermaid

### Q: 如何切换主题？
A: 点击工具栏右侧的 🌙/☀️ 图标，或使用 Ctrl+T

### Q: 文件树不显示？
A: 点击工具栏的"文件"按钮，或使用菜单栏 → 视图 → 切换文件树

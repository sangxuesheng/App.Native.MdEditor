# App.Native.MdEditor - 飞牛 NAS Markdown 编辑器

一个为飞牛 fnOS 设计的现代化 Markdown 编辑器，支持实时预览、GFM、数学公式、流程图等丰富功能。

## 🎯 项目状态

**当前版本**: v1.18.3  
**发布日期**: 2026-03-02  
**开发阶段**: 稳定版本，完美兼容 Edge 和 Safari

## ⌨️ 快捷键

### 文件操作
- `Ctrl/Cmd + S` - 保存文件
- `Ctrl/Cmd + Shift + S` - 另存为
- `Ctrl/Cmd + O` - 打开文件

### 编辑操作
- `Ctrl/Cmd + B` - 加粗
- `Ctrl/Cmd + I` - 斜体
- `Ctrl/Cmd + K` - 插入链接
- `Ctrl/Cmd + Shift + K` - 插入代码块

### 视图操作
- `Ctrl/Cmd + \\` - 切换文件树
- `Ctrl/Cmd + Shift + P` - 切换预览

## 📖 使用指南

### 安装
1. 下载 `App.Native.MdEditor2.fpk`
2. 在飞牛 NAS 应用中心上传安装
3. 启动应用

### 快速开始
1. 点击"新建"创建文件
2. 开始编辑 Markdown
3. 实时查看预览
4. Ctrl+S 保存文件

### 图片上传
1. 复制图片
2. 在编辑器中 Ctrl+V 粘贴
3. 自动上传并插入链接

### 表格插入
1. 点击工具栏"表格"按钮
2. 设置行数和列数
3. 填写表头和内容
4. 点击"插入"

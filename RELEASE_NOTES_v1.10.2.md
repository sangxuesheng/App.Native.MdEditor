# Markdown 编辑器 v1.10.2 发布说明

## 版本信息
- **版本号**: v1.10.2
- **发布日期**: 2025-02-26
- **包大小**: ~50MB

## 🐛 Bug修复

### 主要修复

1. **修复菜单功能失效问题**
   - 修复了v1.10.1中菜单点击无响应的问题
   - 正确实现了menuRef的绑定，使用`display: contents`保持平铺布局
   - 恢复了所有菜单项的正常功能

2. **主题切换功能正常**
   - 确认主题切换功能完整保留
   - 支持三种主题循环切换（浅色/深色/MD3）
   - 在"样式"菜单中可正常切换

### 技术细节

#### 问题原因
- v1.10.1中将MenuBar改为Fragment (`<>`)导致menuRef无法正确绑定
- 点击外部无法检测到，导致菜单无法关闭
- 所有菜单功能受影响

#### 解决方案
- 使用包装div + `display: contents`样式
- 保持平铺布局的同时提供正确的ref绑定点
- 恢复点击外部关闭菜单的功能

## 📦 安装说明

### 方式一：通过应用中心安装
1. 下载 `App.Native.MdEditor2.fpk`
2. 登录飞牛NAS应用中心
3. 点击"本地安装"上传fpk文件
4. 等待安装完成

### 方式二：命令行安装
```bash
# 上传fpk到NAS后
appcenter-cli install-local /path/to/App.Native.MdEditor2.fpk
```

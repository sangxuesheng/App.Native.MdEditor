# Markdown 编辑器 v1.12.5 发布说明

## 发布信息
- **版本号**: 1.12.5
- **发布日期**: 2026-02-27
- **包大小**: 53MB
- **最低系统版本**: 0.9.27

## 主要更新

### 1. 删除上下布局 🗑️
- 移除有问题的上下布局（horizontal）
- 保留稳定的三种布局：
  - 左右布局（vertical）
  - 仅编辑（editor-only）
  - 仅预览（preview-only）
- 解决了上下布局导致的空白页面问题

### 2. 自动保存始终开启 💾
- 自动保存功能默认开启且不可关闭
- 删除所有自动保存的用户设置选项
- 删除工具栏右侧的自动保存按钮和未保存指示器
- 删除菜单中的自动保存选项
- 删除设置对话框中的自动保存设置
- 数据安全优先，自动保护用户工作成果

### 3. 文件树默认关闭 📁
- 打开编辑器时文件树默认关闭
- 提供更大的编辑和预览空间
- 用户可以随时通过工具栏按钮打开/关闭文件树
- 更适合专注于单文件编辑的场景

### 4. Markdown 标题折叠功能 📋
- 实现自定义 Markdown 折叠提供器
- 支持一级到六级标题折叠
- 智能计算折叠范围
- 折叠父级标题会隐藏所有子级内容
- 折叠按钮始终显示
- 支持快捷键操作

### 5. 工具栏样式优化 🎨
- 黑色主题工具栏使用渐变背景
- 从 `#161b22` 到 `#0d1117` 的对角线渐变
- 添加阴影效果，增强层次感
- 与主应用工具栏风格统一

### 6. 行号区域优化 🔢
- 行号宽度从 5 个字符缩窄到 3 个字符
- 节省约 18px 编辑空间
- 行号改为右对齐
- 数字更接近折叠符号
- 视觉上更紧凑整齐

## 折叠功能配置

### 编辑器配置
```javascript
folding: true,
showFoldingControls: 'always',
foldingStrategy: 'auto',
foldingHighlight: true,
foldingMaximumRegions: 5000,
unfoldOnClickAfterEndOfLine: true
```

### 自定义折叠提供器
```javascript
monaco.languages.registerFoldingRangeProvider('markdown', {
  provideFoldingRanges: (model) => {
    // 识别 Markdown 标题
    // 计算折叠范围
    // 返回折叠区域
  }
})
```

## 测试建议

### 测试标题折叠
1. 创建包含多级标题的 Markdown 文档
2. 点击标题左侧的折叠按钮
3. 验证折叠范围正确
4. 测试嵌套折叠

### 测试布局切换
1. 点击布局切换按钮
2. 验证只有 3 种布局
3. 确认无上下布局选项

### 测试文件树
1. 打开编辑器
2. 验证文件树默认关闭
3. 点击文件夹图标打开文件树
4. 再次点击关闭

### 测试自动保存
1. 打开文件
2. 编辑内容
3. 等待自动保存
4. 验证文件已保存
5. 确认无保存按钮/指示器

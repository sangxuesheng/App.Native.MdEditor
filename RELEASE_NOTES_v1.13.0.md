## 🎯 功能特性

### 搜索功能详解

Monaco Editor 提供了强大的内置搜索功能：

1. **基础搜索**
   - 按 `Ctrl/Cmd + F` 打开搜索框
   - 输入搜索文本
   - 使用 `Enter` 查找下一个
   - 使用 `Shift + Enter` 查找上一个
   - 按 `Esc` 关闭搜索框

2. **搜索选项**
   - **Aa** - 大小写敏感
   - **Ab|** - 全词匹配
   - **.*** - 正则表达式

3. **搜索结果**
   - 实时高亮所有匹配项
   - 当前匹配项特殊高亮
   - 显示匹配计数（如：1/5）

### 替换功能详解

1. **基础替换**
   - 按 `Ctrl/Cmd + H` 打开替换框
   - 输入搜索文本和替换文本
   - 点击"替换"按钮替换当前匹配项
   - 点击"全部替换"按钮替换所有匹配项

2. **替换选项**
   - 支持所有搜索选项
   - 支持正则表达式捕获组
   - 替换预览

## 🔧 技术实现

### Monaco Editor 搜索 API

使用 Monaco Editor 的内置搜索功能：

```javascript
// 打开搜索框
editor.trigger('keyboard', 'actions.find')

// 打开替换框
editor.trigger('keyboard', 'editor.action.startFindReplaceAction')
```

### 快捷键绑定

```javascript
// 搜索快捷键
editor.addCommand(monaco.KeyMod.CtrlCmd | monaco.KeyCode.KeyF, () => {
  editor.trigger('keyboard', 'actions.find')
})

// 替换快捷键
editor.addCommand(monaco.KeyMod.CtrlCmd | monaco.KeyCode.KeyH, () => {
  editor.trigger('keyboard', 'editor.action.startFindReplaceAction')
})
```

## 📝 使用示例

### 示例 1: 基础搜索

1. 按 `Ctrl/Cmd + F`
2. 输入 "TODO"
3. 按 `Enter` 查找下一个
4. 按 `Shift + Enter` 查找上一个

### 示例 2: 大小写敏感搜索

1. 按 `Ctrl/Cmd + F`
2. 输入 "markdown"
3. 点击 `Aa` 按钮启用大小写敏感
4. 现在只会匹配 "markdown"，不会匹配 "Markdown"

### 示例 3: 正则表达式搜索

1. 按 `Ctrl/Cmd + F`
2. 点击 `.*` 按钮启用正则表达式
3. 输入 `\d{3}-\d{4}` 查找电话号码格式
4. 按 `Enter` 查找匹配项

### 示例 4: 批量替换

1. 按 `Ctrl/Cmd + H`
2. 搜索框输入 "old_name"
3. 替换框输入 "new_name"
4. 点击"全部替换"按钮
5. 所有 "old_name" 都会被替换为 "new_name"

### 示例 5: 正则表达式替换

1. 按 `Ctrl/Cmd + H`
2. 启用正则表达式模式
3. 搜索框输入 `(\d{4})-(\d{2})-(\d{2})`
4. 替换框输入 `$3/$2/$1`
5. 将日期格式从 "2026-02-27" 转换为 "27/02/2026"

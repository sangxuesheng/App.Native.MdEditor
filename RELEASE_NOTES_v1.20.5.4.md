## 🔧 技术改进

### 高亮文本实现
```javascript
// 预处理转换
let processedContent = content.replace(/==([^=\n]+)==/g, '<mark>$1</mark>')
```

### 文件夹动画
```css
.tree-node-children {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

### 滚动条样式
```css
.preview-pane::-webkit-scrollbar-thumb {
  background: #30363d;
  border-radius: 5px;
  min-height: 30px;
}
```

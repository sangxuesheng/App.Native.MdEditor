## 📖 使用指南

### 访问帮助系统
1. **Markdown 语法帮助**
   - 点击菜单栏 → 帮助 → Markdown 语法
   - 查看完整的语法参考和示例

2. **快捷键列表**
   - 点击菜单栏 → 帮助 → 快捷键列表
   - 查看所有可用的快捷键

3. **关于信息**
   - 点击菜单栏 → 帮助 → 关于
   - 查看版本信息和技术栈

### 使用视图控制
1. **切换工具栏**
   - 点击菜单栏 → 视图 → 切换工具栏
   - 隐藏/显示编辑工具栏

2. **缩放编辑器**
   - 放大：Ctrl++ 或 菜单栏 → 视图 → 放大
   - 缩小：Ctrl+- 或 菜单栏 → 视图 → 缩小
   - 重置：Ctrl+0 或 菜单栏 → 视图 → 重置缩放

3. **切换布局**
   - 点击菜单栏 → 样式 → 选择布局模式
   - 水平布局、垂直布局、仅编辑器、仅预览

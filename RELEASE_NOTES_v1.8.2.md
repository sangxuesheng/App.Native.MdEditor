## 🐛 修复的问题

### 文件树与编辑器重叠 ✅

**问题描述**:
- 文件树（.file-tree）与编辑器的行号区域（.margin-view-overlays）重叠
- 文件树被编辑器覆盖，无法正常显示和操作

**根本原因**:
- 文件树缺少 `position: relative` 定位
- 没有设置 `z-index` 层级
- 导致被编辑器的绝对定位元素覆盖

**修复方案**:
```css
.file-tree {
  width: 280px;
  height: 100%;
  /* ... 其他样式 ... */
  position: relative;  /* 新增 */
  z-index: 10;         /* 新增 */
}
```

**效果**:
- ✅ 文件树正常显示在左侧
- ✅ 不被编辑器覆盖
- ✅ 所有交互功能正常

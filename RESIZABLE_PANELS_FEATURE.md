# 可拉缩面板功能实现报告

## 实现时间
2026-02-26

## 功能概述
为 Markdown 编辑器添加了三个面板（文件树、编辑区、预览区）的可拉缩功能，用户可以通过拖动分隔条自由调整各个区域的大小。

## 实现内容

### 1. 新增组件
创建了 `Resizer` 组件用于处理拖拽调整大小：
- **文件路径**: `src/components/Resizer.jsx`
- **样式文件**: `src/components/Resizer.css`
- **功能**: 
  - 支持垂直和水平方向的拖拽
  - 提供视觉反馈（hover 和 active 状态）
  - 支持三种主题（dark、light、md3）

### 2. 修改的文件

#### App.jsx
- **导入 Resizer 组件**
- **添加状态管理**:
  - `fileTreeWidth`: 文件树宽度（默认 280px，范围 200-600px）
  - `editorWidth`: 编辑器宽度百分比（默认 50%，范围 20%-80%）
  
- **添加 localStorage 持久化**:
  - 自动保存用户调整的面板宽度
  - 下次打开时恢复上次的布局设置

- **添加拖拽处理函数**:
  - `handleFileTreeResize`: 处理文件树宽度调整
  - `handleEditorResize`: 处理编辑器宽度调整

- **修改布局结构**:
  - 在文件树和编辑区之间添加垂直分隔条
  - 在编辑区和预览区之间添加分隔条（根据布局方向自动切换）
  - 使用 inline style 动态设置面板宽度

#### App.css
- **将布局从 Grid 改为 Flexbox**:
  - 更适合动态调整大小
  - 支持灵活的面板宽度控制
  
- **优化布局样式**:
  - `layout-vertical`: 水平排列（flex-direction: row）
  - `layout-horizontal`: 垂直排列（flex-direction: column）
  - 移除了固定的 grid-template 定义

### 3. 功能特性

#### 拖拽调整
- **文件树**: 拖动文件树右侧的分隔条，宽度范围 200-600px
- **编辑器**: 拖动编辑器右侧/下方的分隔条，宽度范围 20%-80%
- **预览区**: 自动占据剩余空间

#### 视觉反馈
- 鼠标悬停时分隔条高亮显示
- 拖拽时光标自动切换（col-resize / row-resize）
- 拖拽时禁用文本选择，提升用户体验

#### 布局适配
- **垂直布局**: 三个面板水平排列，使用垂直分隔条
- **水平布局**: 编辑器和预览区垂直排列，使用水平分隔条
- **仅编辑器**: 文件树和编辑器水平排列
- **仅预览**: 文件树和预览区水平排列

#### 持久化存储
- 使用 localStorage 保存用户偏好
- 键名:
  - `md-editor-filetree-width`: 文件树宽度
  - `md-editor-editor-width`: 编辑器宽度百分比

### 4. 主题支持
分隔条支持三种主题的配色：
- **Dark 主题**: 蓝色高亮 (#58a6ff)
- **Light 主题**: 蓝色高亮 (#0969da)
- **MD3 主题**: 紫色高亮 (#6750a4)

## 使用说明

### 调整文件树宽度
1. 将鼠标移动到文件树右侧边缘
2. 当光标变为 ↔ 时，按住鼠标左键拖动
3. 释放鼠标完成调整

### 调整编辑器宽度
1. 在垂直布局下，将鼠标移动到编辑器右侧边缘
2. 在水平布局下，将鼠标移动到编辑器下方边缘
3. 当光标变为调整大小图标时，拖动调整
4. 释放鼠标完成调整

### 恢复默认布局
如需恢复默认布局，可以清除 localStorage：
```javascript
localStorage.removeItem('md-editor-filetree-width')
localStorage.removeItem('md-editor-editor-width')
```
然后刷新页面。

## 技术细节

### Resizer 组件实现
```javascript
- 使用 useRef 跟踪拖拽状态
- 监听全局 mousemove 和 mouseup 事件
- 通过回调函数传递拖拽偏移量
- 自动清理事件监听器
```

### 宽度计算逻辑
- **文件树**: 固定像素宽度，直接加减 delta
- **编辑器**: 百分比宽度，根据容器总宽度计算百分比变化

### 边界限制
- 使用 Math.max 和 Math.min 限制调整范围
- 防止面板过小或过大影响使用体验

## 🚀 如何使用

### 开发模式

```bash
# 1. 安装依赖
cd app/ui/frontend
npm install

# 2. 启动前端（终端 1）
npm run dev

# 3. 启动后端（终端 2）
node app/server/server.js

# 4. 访问
# http://localhost:3000
```

### 生产模式

```bash
# 1. 构建
./build-frontend.sh

# 2. 启动
node app/server/server.js

# 3. 访问
# http://localhost:18080
```

## 🛠️ 技术方案

### 后端技术栈

```javascript
// 依赖包
{
  "multer": "^1.4.5-lts.1",  // 文件上传
  "sharp": "^0.33.0"          // 图片处理（可选）
}
```

### API 设计

#### 上传图片
```javascript
POST /api/image/upload
Content-Type: multipart/form-data

// 请求
FormData {
  image: File
}

// 响应
{
  "ok": true,
  "data": {
    "url": "/images/2026/02/27/abc123.jpg",
    "thumbnail": "/images/2026/02/27/abc123_thumb.jpg",
    "filename": "abc123.jpg",
    "originalName": "photo.jpg",
    "size": 102400,
    "width": 1920,
    "height": 1080,
    "format": "jpg",
    "uploadDate": "2026-02-27T10:30:00Z"
  }
}
```

#### 获取图片列表
```javascript
GET /api/images?page=1&limit=20&search=keyword

// 响应
{
  "ok": true,
  "data": {
    "images": [...],
    "total": 100,
    "page": 1,
    "limit": 20
  }
}
```

#### 删除图片
```javascript
DELETE /api/image?path=/images/2026/02/27/abc123.jpg

// 响应
{
  "ok": true,
  "message": "图片已删除"
}
```

# 主题导出修复 - 测试指南

## 快速测试步骤

### 1. 准备测试内容
创建一个包含各种元素的测试文档：

```markdown
# 一级标题测试

这是一段普通文本，用于测试段落样式。

## 二级标题测试

### 三级标题测试

**粗体文本** 和 *斜体文本*

> 这是一个引用块
> 用于测试引用样式

- 列表项 1
- 列表项 2
- 列表项 3

\`\`\`javascript
// 代码块测试
function hello() {
  console.log("Hello World");
}
\`\`\`

行内代码：`const x = 10;`

[链接测试](https://example.com)
```

### 2. 测试每个主题

#### 测试 Classic 主题
1. 点击右侧工具栏的"导出配置"按钮
2. 在"主题与基础"中选择"经典"主题
3. 点击"导出"按钮
4. 选择"HTML"格式
5. 导出文件
6. 在浏览器中打开导出的 HTML 文件
7. **验证**：标题应该有渐变色、H2 有背景色和左边框

#### 测试 Elegant 主题
1. 选择"优雅"主题
2. 导出为 HTML
3. **验证**：标题应该有渐变色、引用块有渐变背景和引号装饰

#### 测试 Simple 主题
1. 选择"简洁"主题
2. 导出为 HTML
3. **验证**：H1 有背景色、H2 有左侧色条

#### 测试 Gradient 主题
1. 选择"渐变背景"主题
2. 导出为 HTML
3. **验证**：整个文档有网格背景和渐变光晕效果

#### 测试 Morandi 主题
1. 选择"莫兰迪色系"主题
2. 导出为 HTML
3. **验证**：标题有莫兰迪配色的渐变背景

### 3. 测试主题色功能

1. 选择任意主题（如 Classic）
2. 在"主题色"中选择一个颜色（如经典蓝）
3. 导出为 HTML
4. **验证**：标题、链接等元素应该使用选择的主题色

### 4. 测试 PDF 导出

1. 选择任意主题
2. 导出为 PDF
3. 在打印对话框中选择"另存为 PDF"
4. **验证**：PDF 中的样式应该与预览一致

## 预期结果

### ✅ 修复前的问题
- Classic、Elegant、Simple、Gradient 主题导出后变成默认样式
- 只有 Morandi 主题能正常导出

### ✅ 修复后的效果
- **所有主题**导出后都保持预览时的样式
- HTML 和 PDF 导出都包含完整的主题样式
- 主题色功能正常工作

## 常见问题排查

### 问题：导出的 HTML 还是默认样式
**解决方案：**
1. 清除浏览器缓存
2. 重新启动应用
3. 检查浏览器控制台是否有错误

### 问题：PDF 导出样式不正确
**解决方案：**
1. 确保浏览器允许弹出窗口
2. 等待打印对话框完全加载后再保存
3. 尝试使用不同的浏览器

## 技术验证

### 验证样式是否正确注入
1. 打开导出的 HTML 文件
2. 右键 -> 查看源代码
3. 搜索 `/* 导出配置的主题样式 */`
4. 确认下方有对应主题的 CSS 代码

### 验证代码高亮主题
1. 查看源代码中的 `<link rel="stylesheet" href="...highlight...">`
2. 确认链接指向正确的代码高亮主题

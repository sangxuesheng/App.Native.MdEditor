# 工具栏快捷编辑功能

## ✅ 已完成功能

### 📝 标题插入
- **H1**: 一级标题
- **H2**: 二级标题  
- **H3**: 三级标题
- 自动移除已有标题标记，智能替换

### 🎨 文本格式化
- **B**: 加粗 `**文本**` (快捷键: Ctrl/Cmd + B)
- **I**: 斜体 `*文本*` (快捷键: Ctrl/Cmd + I)
- **S**: 删除线 `~~文本~~`

### 📋 列表插入
- **☰**: 无序列表 `- 项目`
- **≡**: 有序列表 `1. 项目`
- **☑**: 任务列表 `- [ ] 任务`

### 🔗 插入元素
- **🔗**: 插入链接 `[文本](https://)`
- **🖼️**: 插入图片 `![描述](https://)`
- **</>**: 代码块 ` ```代码``` `
- **{`}**: 行内代码 `` `代码` ``

### 📐 其他功能
- **"**: 引用 `> 引用内容`
- **⊞**: 插入表格（3列2行模板）
- **—**: 分隔线 `---`

## 🎯 使用方式

### 1. 包裹选中文本
选中文本后点击按钮，会自动包裹：
- 选中 "重要" → 点击 **B** → `**重要**`
- 选中 "链接" → 点击 🔗 → `[链接](https://)`

### 2. 行首插入
光标在行内，点击列表按钮会在行首插入：
- 点击 ☰ → `- 当前行内容`
- 点击 ≡ → `1. 当前行内容`

### 3. 标题替换
光标在行内，点击标题按钮会智能替换：
- `## 标题` → 点击 H1 → `# 标题`
- `普通文本` → 点击 H2 → `## 普通文本`

### 4. 直接插入
点击表格、分隔线等按钮会直接插入模板

## ⌨️ 快捷键

| 功能 | 快捷键 |
|------|--------|
| 保存 | Ctrl/Cmd + S |
| 加粗 | Ctrl/Cmd + B |
| 斜体 | Ctrl/Cmd + I |
| 插入链接 | Ctrl/Cmd + K |

## 🎨 工具栏样式

- 深色主题，与编辑器风格一致
- 悬停高亮效果（蓝色边框）
- 按钮点击动画
- 响应式设计，移动端自适应
- 横向滚动支持（小屏幕）

## 📦 技术实现

### 核心逻辑
```javascript
handleToolbarInsert(before, after, mode)
```

**模式说明**：
- `wrap`: 包裹选中文本
- `line`: 行首插入
- `heading`: 标题替换
- `insert`: 直接插入

### Monaco Editor 集成
- 使用 `editor.executeEdits()` 执行编辑
- 使用 `editor.setSelection()` 设置光标位置
- 使用 `editor.focus()` 自动聚焦

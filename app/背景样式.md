# 容器背景样式合集
本文档整理了7种不同风格的容器背景CSS样式，所有样式均保留统一的圆角、阴影、内边距核心视觉特征，可根据业务场景直接复制使用。

## 一、基础视觉规范
所有样式共享以下核心属性，保证视觉风格统一：
- `border-radius: 16px`：16px 圆角，弱化尖锐感，提升柔和度
- `box-shadow`：柔和阴影效果，营造轻微悬浮感
- `padding: 40px`：充足内边距，保证容器内内容有足够呼吸空间

## 二、样式列表

### 样式1：基础网格纹理（初始款）
- **风格**：简约肌理感，基础通用
- **适用场景**：后台系统、数据展示、通用功能卡片
```css
.container {
  background-color: #f8f9fa;
  background-image: 
    linear-gradient(rgba(0, 0, 0, 0.05) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0, 0, 0, 0.05) 1px, transparent 1px);
  background-size: 20px 20px;
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;
}
```

### 样式2：渐变光晕背景
- **风格**：柔和高级感，视觉层次丰富
- **适用场景**：表单容器、详情页卡片、品牌展示模块
```css
.container {
  background: linear-gradient(135deg, #f8f9fa 0%, #e9ecef 100%);
  border-radius: 16px;
  /* 双层阴影：外层悬浮+内层微光增强质感 */
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1), 
              inset 0 1px 0 rgba(255, 255, 255, 0.8);
  padding: 40px;
}
```

### 样式3：低饱和度波浪纹理
- **风格**：清新文艺风，纹理轻盈细腻
- **适用场景**：文创类页面、轻量级卡片、个人主页模块
```css
.container {
  background-color: #f8f9fa;
  background-image: 
    repeating-linear-gradient(45deg, rgba(0, 0, 0, 0.03) 0px, rgba(0, 0, 0, 0.03) 2px, 
    transparent 2px, transparent 10px);
  background-size: 28px 28px;
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;
}
```

### 样式4：磨砂玻璃纹理
- **风格**：现代通透感，高级质感
- **适用场景**：叠加在图片/复杂背景上的容器、弹窗、侧边栏
```css
.container {
  background-color: rgba(248, 249, 250, 0.85);
  /* 网点纹理增强磨砂质感 */
  background-image: radial-gradient(rgba(0, 0, 0, 0.02) 1px, transparent 1px);
  background-size: 10px 10px;
  /* 磨砂核心属性（需父级有背景才能体现效果） */
  backdrop-filter: blur(8px);
  -webkit-backdrop-filter: blur(8px); /* 兼容Safari浏览器 */
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;
  /* 细边框增强玻璃通透感 */
  border: 1px solid rgba(255, 255, 255, 0.5);
}
```

### 样式5：极简纯色+微光质感
- **风格**：极致简约，高级哑光感
- **适用场景**：极简UI设计、数据卡片、高端展示模块
```css
.container {
  background: #f8f9fa;
  /* 内层微光渐变提升质感 */
  background-image: linear-gradient(180deg, rgba(255,255,255,0.8) 0%, rgba(255,255,255,0) 100%);
  border-radius: 16px;
  /* 多层阴影增强层次感 */
  box-shadow: 
    0 2px 8px rgba(0,0,0,0.06),
    0 4px 24px rgba(0,0,0,0.1);
  padding: 40px;
  /* 轻微描边增强轮廓 */
  border: 1px solid rgba(0,0,0,0.03);
}
```

### 样式6：复古条纹纹理
- **风格**：轻复古风，温和不刺眼
- **适用场景**：笔记类页面、文档展示、复古风格模块
```css

  background-image: 
    repeating-linear-gradient(

    );
  background-size: 100% 20px;
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;
}
```

### 样式7：科技感点阵纹理
- **风格**：未来简约风，细腻科技感

## 三、使用说明
1. 所有样式的类名均为 `.container`，可根据实际需求修改类名；
2. 磨砂玻璃样式（样式4）需父级元素有背景（图片/渐变/纯色）才能体现磨砂效果；
3. 可根据设计需求调整 `background-size` 改变纹理密度、调整 `rgba` 数值改变纹理深浅；
4. 移动端使用时，建议对 `padding` 做响应式调整（如 `padding: 20px 16px`）。

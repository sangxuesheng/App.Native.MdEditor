### 1. 动画位置上移 20%

### 2. 放大文字和加载点

**文字大小**：
```css

  font-size: clamp(16px, 3vw, 24px); /* 从 10-20px 增大到 16-24px */
}
```

**加载点大小**：
```javascript
// 桌面端和大屏

// 小屏手机（<480px）

}
```

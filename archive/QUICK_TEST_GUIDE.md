### 方法 2: 查看浏览器控制台
1. 打开浏览器控制台（F12）
2. 点击导出按钮
3. 查看调试信息：
   ```

   找到 export-config-styles 元素: true
   主题样式长度: 5000+
   主题样式内容（前 500 字符）: /* 莫兰迪色系 - 全局容器样式 */...
   ```
4. ✅ **主题样式长度应该大于 0**
5. ✅ **主题样式内容应该包含对应主题的 CSS**

### 方法 3: 直接在浏览器中查看
1. 导出 HTML 文件
2. 在浏览器中打开
3. ✅ **样式应该与预览时完全一致**

### 常见问题

**Q: 导出的 HTML 还是默认样式**
A: 
1. 打开控制台查看调试信息
2. 确认"主题样式长度"大于 0
3. 查看导出的 HTML 源代码，确认包含主题样式

**Q: 莫兰迪主题还是不能导出**
A:
1. 确认已经移除了所有 `#preview-area` 选择器
2. 运行：`grep "#preview-area" app/ui/frontend/src/App.jsx`
3. 应该没有任何结果

**Q: 自定义主题不能导出**
A:
1. 确认 `container` 选择器被正确替换为 `.markdown-body`
2. 不要在自定义 CSS 中使用 `#preview-area`

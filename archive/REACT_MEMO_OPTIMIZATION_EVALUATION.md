## 🎯 React.memo 优化分析

### 优化内容

#### 1. 组件 memo 包裹
```javascript
// 识别纯组件并使用 React.memo
const FileTreeItem = React.memo(({ file, onSelect }) => {
  return <div onClick={() => onSelect(file)}>{file.name}</div>
})

const ToolbarButton = React.memo(({ icon, onClick, title }) => {
  return <button onClick={onClick} title={title}>{icon}</button>
})
```

#### 2. useMemo 缓存计算
```javascript
// 缓存复杂计算结果
const processedContent = useMemo(() => {
  return processMarkdown(content)
}, [content])

const filteredFiles = useMemo(() => {
  return files.filter(f => f.name.includes(searchTerm))
}, [files, searchTerm])
```

#### 3. useCallback 缓存函数
```javascript
// 缓存回调函数
const handleFileSelect = useCallback((fileId) => {
  setSelectedFile(fileId)
}, [])

const handleToolbarClick = useCallback((action) => {
  executeAction(action)
}, [])
```

#### 4. 状态优化
- 状态下沉（将只在子组件使用的状态移到子组件）
- 状态拆分（将大对象拆分为多个小状态）
- Context 优化（避免不必要的 Context 更新）

## 二、已实现功能清单

### 2.1 格式分类与常量

**文件**：`app/ui/frontend/src/constants/fileFormats.js`

| 内容 | 说明 |
|------|------|
| 格式分类 | `FORMAT_MD`、`FORMAT_TEXT`、`FORMAT_IMAGE`、`FORMAT_UNSUPPORTED` |
| 扩展名映射 | `EXT_TO_FORMAT`、`EXT_TO_LANGUAGE` |
| 工具函数 | `getFormatFromPath()`、`getLanguageFromPath()`、`getFormatColorClass()`、`isSupportedFormat()` |

**首批支持格式**：
- **MD**：`.md`
- **文本类**：`.text`、`.txt`、`.env`、`.ini`、`.strm`、`.js`、`.ts`、`.html`、`.htm`、`.css`、`.scss`、`.less`、`.json`、`.py`、`.java`、`.c`、`.cpp`、`.cc`、`.cxx`、`.go`、`.rs`、`.php`、`.rb`、`.sh`、`.sql`、`.ml`、`.yaml`、`.yml`、`.vue`、`.pdf`
- **图片类**：`.png`、`.jpg`、`.jpeg`、`.gif`、`.bmp`、`.webp`

---

### 2.2 后端 API

**文件**：`app/server/server.js`

| API | 改动 |
|-----|------|
| `GET /api/files?path=xxx` | 移除 .md 过滤，返回**所有文件**；新增 `size`、`mtime` |
| `GET /api/file?path=xxx&mode=text\|binary\|hex` | 支持 `mode` 参数；text 默认 utf8，binary 返回 base64，hex 返回十六进制 |
| `GET /api/file?path=xxx.pdf` | 自动使用 pdf-parse 提取纯文本 |
| `POST /api/file` | 支持 `encoding: 'base64'` 写入二进制 |

**依赖**：`pdf-parse`（`app/server/package.json`）

---

### 2.3 文件树与列表

**文件**：`app/ui/frontend/src/components/FileTree.jsx`、`FileTree.css`

| 功能 | 实现 |
|------|------|
| 全文件展示 | 依赖 API 变更，自动展示所有格式 |
| 格式着色 | 蓝（MD）、灰（文本）、绿（图片）、红（非支持） |
| 图标扩展 | MD、图片、JSON 等使用对应图标 |
| 格式筛选 | 下拉框「所有文件」/「首批支持格式」 |

---

### 2.4 打开逻辑与自动布局

**文件**：`app/ui/frontend/src/App.jsx`

| 格式 | 默认布局 | Toast 提示 |
|------|----------|------------|
| MD | 保留当前 | 无 |
| 文本类 | 仅编辑 | 「文本类文件默认仅编辑模式打开，可手动切换双栏」 |
| 图片类 | 仅预览 | 「图片类文件默认仅预览模式打开，可手动切换双栏」 |
| 非支持 | 弹窗选择 | 纯文本/Hex/取消 |
| 异常 | 仅编辑 | 「文件读取异常，默认仅编辑模式打开」 |

**非支持格式弹窗**：打开 .exe、.zip 等时，先弹窗选择「纯文本」/「Hex」/「取消」，再加载。

---

### 2.5 编辑器与预览区

| 功能 | 实现 |
|------|------|
| Monaco 语言 | 按扩展名切换（.py→python、.vue→vue 等） |
| MD 预览 | 保持原有 Markdown 富文本渲染 |
| 文本类预览 | 纯文本展示 + highlight.js 语法高亮 |
| 图片预览 | 等比缩放显示，支持点击放大 |
| 非支持预览 | 显示「此格式不支持预览」 |

**图片 Hex 编辑**：
- 切换到仅编辑/双栏时，按需加载 Hex 到编辑区
- 双栏模式下 Hex 编辑后，预览区实时刷新
- 保存时 Hex 转二进制写回原文件

---

### 2.6 保存逻辑

| 格式 | 保存行为 |
|------|----------|
| MD、文本类（非 PDF） | 直接保存 |
| PDF | 弹窗「仅保存提取的纯文本，原 PDF 将被覆盖」→ 确认/另存为 .md/取消 |
| 图片类 | 弹窗「修改图片二进制可能导致损坏」→ 确认/另存为 .md/取消 |
| 非支持 | 弹窗「修改该格式可能导致损坏」→ 确认/另存为 .md/取消 |

**Hex 保存**：Hex 字符串自动转为二进制，通过 `encoding: 'base64'` 写入。

**自动保存**：PDF、图片、非支持格式**不自动保存**，需手动确认。

---

| 功能 | 实现 |

| 图片仅编辑提示 | 切换到仅编辑时 Toast「切换双栏/仅预览可看图片」 |

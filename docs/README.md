# 文档目录

本目录包含项目的历史文档和归档文件。

## 目录结构

```
docs/
├── README.md           # 本文件
└── archive/            # 历史文档归档（230+ 个文件）
    ├── BUILD_*.md      # 构建相关文档
    ├── BUGFIX_*.md     # Bug 修复记录
    ├── FEATURE_*.md    # 功能开发记录
    └── ...             # 其他历史文档
```

## 重要文档位置

### 根目录文档
- **CHANGELOG.md** - 📋 版本更新日志（推荐查看）
- **README.md** - 📖 项目说明
- **START_DEV_MODE.md** - 🚀 开发模式启动指南
- **热更新功能说明.md** - 🔄 热更新详细文档
- **热更新快速参考.md** - ⚡ 热更新快速指南
- **飞牛nas应用开发文档.md** - 📚 飞牛 NAS 开发文档

### 归档文档
`archive/` 目录包含 230+ 个历史文档，主要是开发过程中的：
- 构建报告
- Bug 修复记录
- 功能开发文档
- 测试报告
- 临时笔记

## 查看历史

如需查看某个版本的详细开发记录，请查看 `archive/` 目录中对应的文档。

## 文档命名规范

- `BUILD_*.md` - 构建相关
- `BUGFIX_*.md` - Bug 修复
- `FEATURE_*.md` - 新功能
- `REPORT_*.md` - 报告文档
- `*_v1.x.x.md` - 版本相关

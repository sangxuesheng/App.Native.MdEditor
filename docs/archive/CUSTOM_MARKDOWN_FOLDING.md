## 功能说明
实现了自定义的 Markdown 折叠提供器，支持一级到六级标题的折叠功能。

## 实现方案

### 自定义折叠提供器
在 `handleEditorMount` 函数中注册自定义的 Markdown 折叠提供器：

```javascript
monaco.languages.registerFoldingRangeProvider('markdown', {
  provideFoldingRanges: (model) => {
    const ranges = []
    const lines = model.getLinesContent()
    
    // 查找标题并计算折叠范围
    for (let i = 0; i < lines.length; i++) {
      const line = lines[i]
      const headingMatch = line.match(/^(#{1,6})\s/)
      
      if (headingMatch) {
        const level = headingMatch[1].length
        let endLine = i
        
        // 找到下一个同级或更高级标题
        for (let j = i + 1; j < lines.length; j++) {
          const nextLine = lines[j]
          const nextMatch = nextLine.match(/^(#{1,6})\s/)
          
          if (nextMatch && nextMatch[1].length <= level) {
            endLine = j - 1
            break
          }
          endLine = j
        }
        
        // 只有当有内容可折叠时才添加折叠范围
        if (endLine > i) {
          ranges.push({
            start: i + 1,
            end: endLine + 1,
            kind: monaco.languages.FoldingRangeKind.Region
          })
        }
      }
    }
    
    return ranges
  }
})
```

## 工作原理

### 1. 标题识别
使用正则表达式 `/^(#{1,6}\s/` 识别 Markdown 标题：
- `^` - 行首
- `#{1,6}` - 1到6个 # 号
- `\s` - 后面跟空格

### 2. 层级计算
- 一级标题 `#` - level = 1
- 二级标题 `##` - level = 2
- 三级标题 `###` - level = 3
- 以此类推

### 3. 折叠范围计算
从当前标题开始，查找下一个同级或更高级标题：
- 如果找到同级标题，折叠到前一行
- 如果找到更高级标题，折叠到前一行
- 如果到文件末尾，折叠到最后一行

### 4. 折叠区域生成
只有当标题下有内容时（endLine > i），才创建折叠区域。

### 折叠范围

**一级标题 "# 第一章"**:
- 开始: 第1行
- 结束: 第15行（到 "# 第二章" 之前）
- 包含: 所有内容、二级标题、三级标题

**二级标题 "## 1.1 小节"**:
- 开始: 第5行
- 结束: 第11行（到 "## 1.2 小节" 之前）
- 包含: 小节内容和三级标题

**三级标题 "### 1.1.1 子小节"**:
- 开始: 第9行
- 结束: 第11行（到 "## 1.2 小节" 之前）
- 包含: 子小节内容

**二级标题 "## 1.2 小节"**:
- 开始: 第13行
- 结束: 第15行（到 "# 第二章" 之前）
- 包含: 小节内容

## 边界情况处理

### 1. 空标题
```markdown
# 标题
# 下一个标题
```
第一个标题没有内容，不会显示折叠按钮。

### 2. 文件末尾
```markdown
# 最后一章
内容...
```
折叠到文件末尾。

### 3. 连续标题
```markdown
# 第一章
## 1.1 小节
### 1.1.1 子小节
内容
```
每个标题都可以独立折叠。

### 4. 不规范标题
```markdown
#标题（没有空格）
```
不会被识别为标题，不会折叠。

## 💡 开发经验

### 成功经验
1. **组件化设计**: 将复杂功能拆分为独立组件，便于维护
2. **原生实现**: 不依赖第三方库，减少依赖，提升性能
3. **安全优先**: 从设计阶段就考虑安全性
4. **用户体验**: 提供多种上传方式，满足不同用户习惯

### 遇到的挑战
1. **multipart/form-data 解析**: 手动实现较复杂，需要仔细处理边界
2. **文件读取问题**: Read 工具有时会清空文件，改用 Shell 命令
3. **主题适配**: 需要为三种主题分别编写样式

### 解决方案
1. 使用 Buffer 操作处理二进制数据
2. 使用 git restore 恢复文件，使用 Shell 命令修改
3. 使用 CSS 变量和 data-theme 属性统一管理主题

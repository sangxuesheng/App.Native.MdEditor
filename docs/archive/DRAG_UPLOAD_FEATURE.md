# v1.20.0 拖拽上传图片功能说明

## 功能概述
v1.20.0 版本新增了拖拽上传图片功能，用户可以直接将图片文件拖拽到编辑器中上传并插入 Markdown 图片链接。

## 主要特性

### 1. 拖拽上传
- 支持从文件管理器直接拖拽图片到编辑器
- 自动识别图片文件类型（jpg, png, gif, webp 等）
- 支持多张图片同时拖拽上传

### 2. 视觉反馈
- 拖拽悬停时编辑器显示半透明效果
- 显示"拖放图片到此处上传"提示文字
- 适配所有主题（深色/浅色/Material Design 3）

### 3. 智能插入
- 在鼠标拖放位置插入图片链接
- 如果无法获取位置，则在当前光标位置插入
- 多张图片自动换行排列

### 4. 状态提示
- 上传过程中显示进度提示
- 上传成功显示成功消息
- 上传失败显示错误提示

## 使用方法

### 基本使用
1. 打开 Markdown 文件
2. 从文件管理器选择一张或多张图片
3. 拖拽到编辑器区域
4. 松开鼠标，图片自动上传并插入链接

### 支持的图片格式
- JPEG/JPG
- PNG
- GIF
- WebP
- 其他浏览器支持的图片格式

### 示例效果
拖拽 3 张图片后，编辑器会自动插入：
```markdown
![image1.jpg](/uploads/images/image1.jpg)
![image2.png](/uploads/images/image2.png)
![image3.gif](/uploads/images/image3.gif)
```

## 技术实现

### 事件监听
- `dragover`: 显示拖拽悬停效果
- `dragleave`: 移除悬停效果
- `drop`: 处理文件上传和插入

### API 调用
- 使用 `/api/image/upload` 接口上传图片
- 支持 FormData 多文件上传
- 返回图片 URL 和文件名

### 样式适配
- 深色主题：蓝色边框和提示
- 浅色主题：深蓝色边框和提示
- MD3 主题：紫色边框和提示

## 与现有功能的配合

### 粘贴上传
- 保留原有的粘贴图片功能
- 拖拽和粘贴可以混合使用

### 图片管理
- 上传的图片可在图片管理器中查看
- 支持删除和预览

### 自动保存
- 插入图片后自动触发保存
- 保证内容不丢失

## 注意事项

1. **文件大小限制**：根据服务器配置，可能有单个文件大小限制
2. **网络连接**：需要网络连接才能上传图片
3. **权限要求**：需要有上传图片的权限

# ✅ 编辑器行号区域背景色

## 完成时间
2026-02-27 01:55

## 修改内容

### 功能说明
为 Monaco Editor 的行号区域（margin）添加背景色，在不同主题下提供更好的视觉层次感。

## 添加的样式

### 深色主题
```css
.theme-dark .monaco-editor .margin,
.theme-dark .monaco-editor .margin-view-overlays {
  background: #161b22 !important;
}
```

**颜色**: `#161b22`（深灰蓝色）
**效果**: 与工具栏渐变背景的起始色一致

### 浅色主题
```css
.theme-light .monaco-editor .margin,
.theme-light .monaco-editor .margin-view-overlays {
  background: #f6f8fa !important;
}
```

**颜色**: `#f6f8fa`（浅灰色）
**效果**: 与浅色主题的整体背景色一致

### MD3 主题
```css
.theme-md3 .monaco-editor .margin,
.theme-md3 .monaco-editor .margin-view-overlays {
  background: linear-gradient(135deg, #f6edff 0%, #fef7ff 100%) !important;
}
```

**颜色**: 渐变（`#f6edff` → `#fef7ff`）
**效果**: 与 MD3 主题的紫色渐变风格一致

## 技术实现

### CSS 选择器
```css
.theme-dark .monaco-editor .margin,
.theme-dark .monaco-editor .margin-view-overlays
```

**说明**:
- `.margin`: 行号区域容器
- `.margin-view-overlays`: 行号内容覆盖层
- 两者都需要设置背景色以确保完整覆盖

### !important 的使用
```css
background: #161b22 !important;
```

**原因**:
- Monaco Editor 有内联样式
- 需要 `!important` 覆盖默认样式
- 确保背景色生效

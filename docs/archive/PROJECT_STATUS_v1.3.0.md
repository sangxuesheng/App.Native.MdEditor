## 🐛 已知问题

### 1. PDF 导出
- **问题**: 需要服务器端支持
- **影响**: 中等
- **临时方案**: 使用浏览器打印功能
- **计划**: v2.0 添加服务器端生成

### 2. 大文件性能
- **问题**: 部分 JS chunk 超过 500KB
- **影响**: 初始加载时间较长
- **计划**: 代码分割优化

### 3. 源文件管理
- **问题**: 开发时文件可能被清空
- **影响**: 低（已有备份）
- **解决**: 使用版本控制

# 快速开始指南 v1.3.0

## 🚀 5分钟快速部署

### 前置条件
- ✅ 飞牛 NAS 系统（fnOS 0.9.27+）
- ✅ Node.js v22 运行时已安装
- ✅ fnpack 工具

---

## 📦 方式一：使用自动化脚本（推荐）

### 1. 一键部署
```bash
cd /vol4/1000/开发文件夹/mac/开发/App.Native.MdEditor
./deploy.sh
```

这个脚本会自动：
- ✅ 构建前端
- ✅ 打包 FPK
- ✅ 显示部署信息

### 2. 安装应用
```bash
# 复制生成的 .fpk 文件到 NAS
# 然后执行：
appcenter-cli install-fpk App.Native.MdEditor2.fpk
```

---

## 🔧 方式二：手动部署

### 步骤 1: 构建前端
```bash
export PATH=/var/apps/nodejs_v22/target/bin:$PATH
cd app/ui/frontend
npm run build
```

### 步骤 2: 打包 FPK
```bash
cd ../../..
../fnpack-1.2.1-darwin-arm64 build
```

### 步骤 3: 安装
```bash
appcenter-cli install-fpk App.Native.MdEditor2.fpk
```

---

## 🎯 首次使用

### 1. 打开应用
- 在飞牛 NAS 桌面找到"Markdown 编辑器 v2"
- 点击图标启动

### 2. 创建第一个文件
1. 点击工具栏"新建"按钮
2. 选择模板（推荐：笔记）
3. 选择保存位置
4. 输入文件名
5. 点击"创建"

### 3. 开始编辑
- 左侧：文件树
- 中间：编辑器
- 右侧：实时预览

## ✨ 核心功能

### 📝 编辑功能
- **快捷键**:
  - `Ctrl/Cmd + S`: 保存
  - `Ctrl/Cmd + B`: 加粗
  - `Ctrl/Cmd + I`: 斜体
  - `Ctrl/Cmd + K`: 插入链接

### 💾 文件操作
- **新建**: 支持 6 种模板
- **保存**: 自动保存（30秒）
- **另存为**: 保存到新位置
- **导出**: HTML/Markdown/TXT

### ⚙️ 设置
- **主题**: 深色/浅色
- **自动保存**: 可自定义间隔
- **编辑器**: 字体、行高、Tab 等

### 🎨 布局
- **水平布局**: 上下分屏
- **垂直布局**: 左右分屏
- **仅编辑器**: 专注写作
- **仅预览**: 查看效果

## 🐛 常见问题

### Q: 应用无法启动？
**A**: 检查 Node.js v22 是否已安装
```bash
appcenter-cli list-apps | grep nodejs
```

### Q: 文件无法保存？
**A**: 检查目录权限，确保应用有写入权限

### Q: 预览不显示？
**A**: 刷新页面或切换布局模式

### Q: 如何导出 PDF？
**A**: 
1. 导出为 HTML
2. 在浏览器中打开
3. 使用浏览器的"打印为 PDF"功能

## 💡 使用技巧

### 1. 快速插入
使用工具栏按钮快速插入：
- 标题（H1-H6）
- 列表（有序/无序）
- 链接和图片
- 代码块
- 表格

### 2. 自动保存
- 默认 30 秒自动保存
- 可在设置中调整间隔
- 手动保存：`Ctrl/Cmd + S`

### 3. 草稿恢复
- 意外关闭后自动提示恢复
- 可选择恢复或丢弃草稿

### 4. 主题切换
- 点击工具栏 🌙/☀️ 图标
- 或在设置中选择

### 5. 文件管理
- 使用文件树浏览
- 支持搜索和过滤
- 点击文件快速打开

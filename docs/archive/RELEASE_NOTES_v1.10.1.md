# Markdown 编辑器 v1.10.1 发布说明

## 版本信息
- **版本号**: v1.10.1
- **发布日期**: 2025-02-25
- **包大小**: ~52MB

## 🎨 UI优化更新

### 主要改进

1. **菜单栏布局优化**
   - 移除嵌套的菜单栏容器，实现平铺结构
   - 菜单项直接集成在工具栏中，界面更加整洁
   - 优化菜单栏CSS，减少不必要的DOM层级

2. **自动保存功能调整**
   - 将独立的"自动保存"按钮移至"文件"菜单
   - 在文件菜单中添加"自动保存: 开启/关闭"选项
   - 保持功能完整性的同时简化工具栏布局

3. **未保存指示器优化**
   - 为未保存指示器（●）添加鼠标悬停提示
   - 显示"有未保存的更改"文字说明
   - 提升用户体验

4. **文件树与编辑器对齐修复**
   - 修复文件树搜索框与编辑器工具栏的重叠问题
   - 统一文件树header和编辑器工具栏的高度（53px）
   - 调整padding和gap确保垂直对齐
   - 修正文件树宽度配置（280px），消除30px重叠

### 技术细节

#### 修改的文件
- `App.jsx` - 添加autoSave props，移除自动保存按钮，修正grid布局宽度
- `App.css` - 将文件树宽度从250px统一为280px
- `MenuBar.jsx` - 改为Fragment结构，添加自动保存菜单项
- `MenuBar.css` - 删除`.menubar`容器样式
- `FileTree.css` - 调整header padding为8px 12px，gap为6px
- `EditorToolbar.css` - 调整padding为8px 12px，min-height为53px

#### 布局改进
- 文件树宽度: 280px（统一）
- 编辑器工具栏高度: 53px
- 文件树header高度: 53px
- 完美对齐，无重叠

## 📦 安装说明

### 方式一：通过应用中心安装
1. 下载 `App.Native.MdEditor2.fpk`
2. 登录飞牛NAS应用中心
3. 点击"本地安装"上传fpk文件
4. 等待安装完成

### 方式二：命令行安装
```bash
# 上传fpk到NAS后
appcenter-cli install-local /path/to/App.Native.MdEditor2.fpk
```

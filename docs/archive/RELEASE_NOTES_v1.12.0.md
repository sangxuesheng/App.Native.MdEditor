## 🎯 使用指南

### 如何使用右键菜单

1. **打开右键菜单**：
   - 在文件树中右键点击文件或文件夹
   - 在空白区域右键点击（显示根目录菜单）

2. **重命名文件/文件夹**：
   - 右键点击 → 选择"重命名"
   - 输入新名称 → 点击"确定"

3. **删除文件/文件夹**：
   - 右键点击 → 选择"删除"
   - 确认删除操作

4. **复制/粘贴文件**：
   - 右键点击源文件 → 选择"复制"或"剪切"
   - 右键点击目标文件夹 → 选择"粘贴"

5. **新建文件夹**：
   - 右键点击父文件夹 → 选择"新建文件夹"
   - 输入文件夹名称 → 点击"创建"

6. **添加到收藏夹**：
   - 右键点击文件/文件夹 → 选择"添加到收藏夹"
   - 再次点击可取消收藏

### 快捷操作

- **ESC 键**：关闭对话框
- **Enter 键**：确认操作
- **点击外部**：关闭右键菜单

## 🐛 已知问题

### 1. 剪贴板限制
- 使用 localStorage 模拟剪贴板
- 刷新页面后剪贴板内容会丢失
- 不支持跨浏览器标签页复制

### 2. 大文件夹操作
- 复制/移动大文件夹可能需要较长时间
- 建议分批处理大量文件

### 3. 文件名限制
- 不支持特殊字符：`< > : " / \ | ? *`
- 文件夹名称不能以点开头

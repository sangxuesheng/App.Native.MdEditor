## 🎉 新增功能

### 🖼️ 完整的图片处理系统

#### 1. 图片上传
- ✅ **拖拽上传**: 直接拖拽图片文件到上传区域
- ✅ **点击上传**: 通过文件选择对话框上传
- ✅ **粘贴上传**: 从剪贴板直接粘贴图片（Ctrl+V）
- ✅ **多文件上传**: 支持一次上传多张图片
- ✅ **格式支持**: JPG, PNG, GIF, WebP
- ✅ **大小限制**: 单个文件最大 10MB
- ✅ **上传进度**: 实时显示上传进度

#### 2. 图片管理器
- ✅ **图片列表**: 网格布局展示所有已上传图片
- ✅ **搜索功能**: 按文件名搜索图片
- ✅ **分页浏览**: 每页显示 20 张图片
- ✅ **图片信息**: 显示文件名、大小、上传日期
- ✅ **快速操作**: 插入、复制链接、删除
- ✅ **批量管理**: 支持批量选择和删除

#### 3. 图片预览
- ✅ **全屏预览**: 点击图片查看大图
- ✅ **缩放功能**: 鼠标滚轮缩放（50% - 300%）
- ✅ **缩放控制**: 放大、缩小、重置按钮
- ✅ **图片信息**: 显示详细的文件信息
- ✅ **快捷操作**: 复制链接、插入、下载、删除

#### 4. 编辑器集成
- ✅ **工具栏按钮**: 点击图片按钮打开图片管理器
- ✅ **自动插入**: 选择图片后自动插入 Markdown 语法
- ✅ **语法格式**: `![文件名](图片URL)`

#### 5. 后端 API
- ✅ **上传 API**: `POST /api/image/upload`
- ✅ **列表 API**: `GET /api/images`
- ✅ **删除 API**: `DELETE /api/image`
- ✅ **静态服务**: `/uploads/` 路径提供图片访问

## 🎯 使用方法

### 上传图片

**方法 1: 工具栏按钮**
1. 点击工具栏的图片按钮 🖼️
2. 在图片管理器中点击"上传图片"
3. 选择或拖拽图片文件

**方法 2: 直接粘贴**
1. 复制图片到剪贴板
2. 在上传区域按 Ctrl+V
3. 图片自动上传

**方法 3: 拖拽上传**
1. 从文件管理器拖拽图片
2. 放到上传区域
3. 自动开始上传

### 插入图片

**方法 1: 图片管理器**
1. 点击工具栏图片按钮
2. 在图片列表中点击"➕"按钮
3. 图片自动插入到编辑器

**方法 2: 预览窗口**
1. 点击图片查看大图
2. 点击"插入"按钮
3. 图片自动插入

### 管理图片

**查看图片**
- 点击图片缩略图查看大图
- 使用鼠标滚轮缩放
- 点击缩放按钮调整大小

**删除图片**
- 在图片卡片上点击 🗑️ 按钮
- 或在预览窗口点击"删除"
- 确认后永久删除

**复制链接**
- 点击 📋 按钮复制图片 URL
- 可用于分享或外部引用

## 🔧 配置说明

### 上传限制
- **文件大小**: 10MB（可在 server.js 中修改）
- **文件类型**: JPG, PNG, GIF, WebP
- **存储位置**: `app/uploads/images/`

### API 端点
```javascript
// 上传图片
POST /api/image/upload
Content-Type: multipart/form-data

// 获取图片列表
GET /api/images?page=1&limit=20&search=keyword

// 删除图片
DELETE /api/image?url=/uploads/images/2026/02/27/xxx.jpg

// 访问图片
GET /uploads/images/2026/02/27/xxx.jpg
```

## ✨ 新功能

### 1. 另存为功能 🆕

将当前文档保存到新位置，支持：
- 📁 选择保存目录
- ✏️ 自定义文件名
- ⚠️ 文件冲突检测
- 🔄 覆盖确认对话框

**使用方法**:
1. 点击工具栏"另存为"按钮
2. 选择保存位置和文件名
3. 确认保存

---

### 2. 导出功能 🆕

支持多种格式导出：

#### 📄 HTML 导出
- 支持 GitHub Dark/Light 主题
- 可选包含 CSS 样式
- 完整的 Markdown 渲染
- 支持 LaTeX 和 Mermaid

#### 📝 Markdown 导出
- 导出原始 Markdown 文件
- 保留所有格式

#### 📃 TXT 导出
- 导出为纯文本
- 适合简单分享

#### 📄 PDF 导出
- 提示使用浏览器"打印为 PDF"功能
- 保持完整格式

**使用方法**:
1. 点击工具栏"导出"按钮
2. 选择导出格式
3. 配置导出选项
4. 点击"导出"下载文件

---

### 3. 设置对话框 🆕

完整的应用配置管理：

#### 通用设置
- 🎨 主题选择（深色/浅色）

#### 自动保存设置
- ⚡ 开启/关闭自动保存
- ⏱️ 自定义保存间隔（10-300秒）

#### 编辑器设置
- 📏 字体大小（10-24px）
- 📐 行高（16-40px）
- ⌨️ Tab 大小（2-8空格）
- 🔤 字体选择（5种专业编程字体）
- 📄 自动换行
- 🔢 显示行号
- 🗺️ 显示小地图

#### 配置持久化
- 💾 设置自动保存到浏览器
- 🔄 一键恢复默认设置

**使用方法**:
1. 点击工具栏"⚙️"按钮
2. 调整各项设置
3. 点击"保存"应用

## 🐛 已知问题

### PDF 导出
- **问题**: 需要服务器端支持
- **临时方案**: 使用浏览器"打印为 PDF"功能
- **计划**: 未来版本添加服务器端 PDF 生成

### 大文件性能
- **问题**: 部分 JS chunk 超过 500KB
- **影响**: 初始加载时间较长
- **计划**: 优化代码分割

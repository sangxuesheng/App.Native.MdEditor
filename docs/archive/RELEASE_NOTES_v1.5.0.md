# Markdown 编辑器 v1.5.0 发布说明

## 📅 发布日期
2026-02-24

## 🎉 版本亮点

### 专业菜单栏系统
v1.5.0 的核心更新是引入了**专业的一二级菜单栏**，提供更符合桌面应用习惯的操作体验。

## ✨ 新增功能

### 1. 专业菜单栏（MenuBar）
- ✅ **文件菜单**
  - 新建文件（Ctrl+N）
  - 保存（Ctrl+S）
  - 另存为（Ctrl+Shift+S）
  - 导出子菜单（公众号格式、HTML、Markdown等）

- ✅ **编辑菜单**
  - 撤销/重做（Ctrl+Z/Y）
  - 复制/粘贴（Ctrl+C/V）
  - 格式化文档（Shift+Alt+F）
  - 查找/替换（Ctrl+F/H）

- ✅ **格式菜单**
  - 标题子菜单（一至六级标题，Ctrl+1-6）
  - 文本格式（加粗、斜体、删除线）
  - 列表（无序、有序、任务列表）
  - 引用和代码块

- ✅ **插入菜单**
  - 链接（Ctrl+K）
  - 图片
  - 表格
  - 分隔线
  - 代码块、数学公式、Mermaid 图表

- ✅ **样式菜单**
  - 布局切换（水平、垂直、仅编辑器、仅预览）
  - 主题切换（Ctrl+T）

- ✅ **视图菜单**
  - 切换文件树（Ctrl+B）
  - 切换工具栏
  - 缩放控制（Ctrl+/- /0）

- ✅ **帮助菜单**
  - Markdown 语法
  - 快捷键列表
  - 关于

### 2. 菜单栏特性

## 📖 使用指南

### 菜单栏使用
1. **点击菜单名称**打开下拉菜单
2. **鼠标悬停**在有子菜单的项目上查看子菜单
3. **点击菜单项**执行对应操作
4. **点击外部**或按 ESC 关闭菜单

### 快捷键
所有快捷键在菜单中都有显示，主要快捷键：
- `Ctrl+N`: 新建文件
- `Ctrl+S`: 保存
- `Ctrl+Shift+S`: 另存为
- `Ctrl+Z/Y`: 撤销/重做
- `Ctrl+F/H`: 查找/替换
- `Ctrl+B`: 加粗
- `Ctrl+I`: 斜体
- `Ctrl+K`: 插入链接
- `Ctrl+T`: 切换主题
- `Ctrl+1-6`: 插入标题

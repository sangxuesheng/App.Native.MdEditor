## 🎯 核心功能

### 1. 最近文件列表
**位置**: 文件菜单 → 最近文件

#### 使用场景
- 快速访问最近编辑的文件
- 查看文件访问历史
- 提高工作效率

#### 特性
- 自动记录最近打开的 20 个文件
- 显示相对时间（刚刚/2分钟前/1小时前/昨天）
- 点击快速打开
- 支持清空列表

### 2. 收藏夹
**位置**: 文件树顶部

#### 使用场景
- 收藏常用文件和文件夹
- 快速访问重要内容
- 组织工作文件

#### 特性
- 支持收藏文件和文件夹（最多 50 个）
- 可折叠/展开面板
- 显示收藏数量
- 悬停显示移除按钮
- 支持清空收藏夹

### 3. 增强搜索
**位置**: 文件树搜索框

#### 使用场景
- 快速查找文件
- 模糊搜索文件名
- 查看搜索历史

#### 特性
- 模糊搜索算法（支持 "rdm" 匹配 "README.md"）
- 搜索历史记录（最多 20 条）
- 搜索结果高亮显示
- 键盘快捷键（Enter 搜索，Esc 关闭）

## 🔧 技术架构

### 数据持久化
所有功能使用 localStorage 存储：

```javascript
{
  // 最近文件
  "md-editor-recent-files": [
    { path: "/path/to/file.md", timestamp: 1708819200000, name: "file.md" }
  ],
  
  // 收藏夹
  "md-editor-favorites": [
    { path: "/path/to/file.md", type: "file", name: "file.md", timestamp: 1708819200000, order: 0 }
  ],
  
  // 搜索历史
  "md-editor-search-history": [
    "readme", "app.jsx", "config"
  ]
}
```

### 工具类
- `recentFilesManager.js` - 最近文件管理
- `favoritesManager.js` - 收藏夹管理
- `fileSearcher.js` - 搜索算法和历史管理

### 组件
- `FavoritesPanel.jsx` - 收藏夹面板
- `FileSearchBox.jsx` - 增强搜索框

## 📖 使用指南

### 最近文件列表
1. 打开任意文件，自动记录到列表
2. 点击"文件"菜单 → "最近文件"
3. 点击文件名快速打开
4. 点击"清空列表"清除历史

### 收藏夹
1. 在文件树中找到要收藏的文件
2. 右键点击（未来功能）或使用快捷键
3. 文件出现在收藏夹面板
4. 点击收藏项快速打开
5. 悬停显示 × 按钮，点击移除

### 增强搜索
1. 在文件树搜索框输入关键词
2. 支持模糊搜索（如 "rdm" 匹配 "README.md"）
3. 按 Enter 保存到搜索历史
4. 点击搜索框显示历史记录
5. 点击历史记录快速搜索

## 🛠️ 技术方案

### 后端 API 设计

#### 创建文件
```javascript
POST /api/file/create
{
  "path": "/path/to/new/file.md",
  "content": ""
}
```

#### 创建文件夹
```javascript
POST /api/folder/create
{
  "path": "/path/to/new/folder"
}
```

#### 重命名
```javascript
POST /api/file/rename
{
  "oldPath": "/path/to/old.md",
  "newPath": "/path/to/new.md"
}
```

#### 删除
```javascript
DELETE /api/file?path=/path/to/file.md
DELETE /api/folder?path=/path/to/folder
```

#### 移动/复制
```javascript
POST /api/file/move
{
  "sourcePath": "/path/to/source.md",
  "targetPath": "/path/to/target.md"
}

POST /api/file/copy
{
  "sourcePath": "/path/to/source.md",
  "targetPath": "/path/to/copy.md"
}
```

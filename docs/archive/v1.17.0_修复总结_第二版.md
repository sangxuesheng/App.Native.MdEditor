```javascript
useEffect(() => {
  const handleGlobalPaste = async (e) => {
    if (!editorRef.current) return
    
    const items = e.clipboardData?.items
    if (!items) return
    
    for (let i = 0; i < items.length; i++) {
      const item = items[i]
      if (item.type.startsWith('image/')) {
        e.preventDefault()
        e.stopPropagation()
        
        let file = item.getAsFile()
        // ... 上传逻辑
      }
    }
  }
  
  window.addEventListener('paste', handleGlobalPaste)
  
  return () => {
    window.removeEventListener('paste', handleGlobalPaste)
  }
}, [])
```

## 💡 使用说明

### 粘贴图片上传
1. 截图或复制图片文件
2. 在编辑器中按 Ctrl+V（Mac: Cmd+V）
3. 等待上传完成
4. 自动插入 `![图片](URL)` 语法

### 调整压缩设置
1. 菜单栏 → 视图 → 设置
2. 滚动到"图片"区域
3. 调整压缩参数：
   - 自动压缩：开/关
   - 压缩质量：50%-100%
   - 最大宽度：800-4096px
   - 最大高度：600-4096px
4. 点击"保存"

### 预览区查看图片
1. 在编辑器中插入图片
2. 预览区自动渲染
3. 点击图片放大查看
4. 查看详细信息或下载

---

## ⚠️ 注意事项

1. **粘贴功能**: 只支持粘贴图片，不支持粘贴图片URL
2. **压缩设置**: 修改后需重新上传图片才生效
3. **预览功能**: 只在预览区有效，编辑器中的图片语法不可点击
4. **批量删除**: 操作不可撤销，请谨慎使用
5. **图片存储**: 图片保存在 `@appshare/mdeditor/images` 目录

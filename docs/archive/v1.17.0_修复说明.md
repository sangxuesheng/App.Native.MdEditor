# v1.17.0 修复说明

## 修复的问题

### 1. ✅ 剪贴板图片粘贴功能修复
**问题**: Ctrl+V 无法粘贴剪贴板中的图片到编辑器

**修复方案**:
- 改用事件捕获阶段监听粘贴事件（`addEventListener('paste', handler, true)`）
- 在 Monaco Editor 拦截之前捕获图片粘贴事件
- 阻止默认行为并自动上传图片

**测试方法**:
1. 复制一张图片到剪贴板（截图或复制图片文件）
2. 在编辑器中按 Ctrl+V
3. 应该自动上传并插入图片 Markdown 语法

---

### 2. ✅ 图片压缩设置功能
**问题**: 用户无法控制是否压缩图片和压缩比例

**新增功能**:
- 在设置对话框中添加"图片"设置区域
- 可开启/关闭自动压缩
- 可调整压缩质量（50%-100%）
- 可设置最大宽度（800-4096px）
- 可设置最大高度（600-4096px）
- 设置保存在 localStorage 中

**默认设置**:
- 自动压缩：开启
- 压缩质量：80%
- 最大宽度：1920px
- 最大高度：1080px

**测试方法**:
1. 点击菜单栏"设置"按钮
2. 滚动到"图片"设置区域
3. 调整压缩设置
4. 点击"保存"
5. 上传图片测试压缩效果

---

### 3. ✅ 预览区图片点击预览
**问题**: 预览区的图片无法点击放大查看

**新增功能**:
- 预览区所有图片自动添加点击事件
- 点击图片弹出预览对话框
- 显示图片详细信息
- 支持下载图片

**测试方法**:
1. 在编辑器中插入图片：`![测试](图片URL)`
2. 在右侧预览区查看渲染的图片
3. 点击预览区的图片
4. 应该弹出图片预览对话框

---

### 4. ✅ 批量管理按钮位置调整
**问题**: 批量管理按钮在顶部，不够醒目

**优化方案**:
- 将批量管理按钮移到图片库浮窗左下角
- 固定定位，始终可见
- 半透明背景，不遮挡内容
- 选择模式下显示全选、删除、取消按钮

**测试方法**:
1. 打开图片管理对话框
2. 切换到"图片库"标签页
3. 在左下角应该看到"批量管理"按钮
4. 点击后显示全选、删除选中、取消按钮

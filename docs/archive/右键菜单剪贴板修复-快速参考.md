# 右键菜单剪贴板功能修复 - 快速参考

## 问题
开发模式下右键菜单的剪切、复制、粘贴功能正常，生产模式下不可用。

## 原因
Clipboard API 在非 HTTPS 环境下受限，原有 fallback 机制不完善。

## 解决方案
✅ 添加多层 fallback 机制
✅ 增强剪贴板状态检测
✅ 支持所有环境（HTTP/HTTPS）

## 技术细节

### Fallback 层级
1. **第一层**: `navigator.clipboard` API（现代浏览器 + HTTPS）
2. **第二层**: Monaco Editor 内置命令（所有环境）
3. **第三层**: `document.execCommand`（旧版浏览器）

### 关键代码
```javascript
// 复制操作示例
case 'copy':
  try {
    if (navigator.clipboard && navigator.clipboard.writeText) {
      await navigator.clipboard.writeText(text)
    } else {
      editor.trigger('keyboard', 'editor.action.clipboardCopyAction')
    }
  } catch (error) {
    editor.trigger('keyboard', 'editor.action.clipboardCopyAction')
  }
  break
```

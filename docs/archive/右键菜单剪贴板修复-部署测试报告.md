## 修复内容

### 核心修改
1. **完全手动实现剪贴板操作**
   - 不依赖 `editor.trigger()` 命令
   - 不依赖 `document.execCommand()`
   - 直接使用 Monaco Editor API

2. **剪切操作**
   - 使用 `navigator.clipboard.writeText()` 写入剪贴板
   - 使用 `editor.executeEdits()` 删除选中内容
   - 正确设置光标位置

3. **复制操作**
   - 使用 `navigator.clipboard.writeText()` 写入剪贴板
   - 不修改编辑器内容

4. **粘贴操作**
   - 使用 `navigator.clipboard.readText()` 读取剪贴板
   - 使用 `editor.executeEdits()` 插入内容
   - 正确计算多行文本的光标位置

### 降级方案
- 主方案：Clipboard API
- 降级方案：隐藏 textarea + execCommand
- 最终方案：提示用户使用快捷键

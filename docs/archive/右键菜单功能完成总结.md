## 📋 集成步骤

### 快速集成（5步）

1. **查看集成代码**
   ```bash
   cat src/EditorContextMenuIntegration.jsx
   ```

2. **运行集成脚本**
   ```bash
   bash integrate-context-menu.sh
   ```

3. **导入组件**（在 App.jsx 顶部）
   ```javascript
   import EditorContextMenu from './components/EditorContextMenu'
   ```

4. **添加状态**（在 useState 区域）
   ```javascript
   const [contextMenu, setContextMenu] = useState(null)
   const [clipboardContent, setClipboardContent] = useState(null)
   ```

5. **添加菜单组件**（在 JSX 的 return 末尾）
   ```javascript
   {contextMenu && (
     <EditorContextMenu
       x={contextMenu.x}
       y={contextMenu.y}
       selectedText={contextMenu.selectedText}
       selectedImage={contextMenu.selectedImage}
       theme={editorTheme}
       clipboardHasContent={!!clipboardContent}
       onAction={handleContextMenuAction}
       onClose={() => setContextMenu(null)}
     />
   )}
   ```

## ⚠️ 注意事项

1. **图片删除 API**
   - 需要后端提供 `/api/image/delete` 接口
   - 仅对本地上传的图片启用删除功能

2. **剪贴板权限**
   - 某些浏览器需要用户授权
   - 提供了降级方案（使用 Monaco Editor 内置方法）

3. **性能优化**
   - 使用 `useCallback` 避免不必要的重渲染
   - 事件监听器在组件卸载时自动清理

4. **兼容性**
   - 支持现代浏览器（Chrome, Firefox, Safari, Edge）
   - 需要 React 18.3.1+
   - 需要 Monaco Editor

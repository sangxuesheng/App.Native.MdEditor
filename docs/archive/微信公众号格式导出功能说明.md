- 使用 `Clipboard API` 的 `write()` 方法

- 主方案：`navigator.clipboard.write()` (富文本)
- 降级方案：`document.execCommand('copy')` (选中复制)
- 确保在任何环境下都能工作

## 技术实现

### 1. 转换函数 `convertToWechatFormat()`

```javascript
const convertToWechatFormat = (html) => {
  // 创建临时容器
  const container = document.createElement('div')
  container.innerHTML = html
  
  // 微信公众号样式
  const wechatStyles = `
    <style>
      /* 基础样式 */
      .wechat-content {
        font-size: 16px;
        line-height: 1.75;
        color: #333;
      }
      
      /* 标题样式 */
      .wechat-content h1 {
        font-size: 24px;
        border-bottom: 2px solid #3daeff;
      }
      
      .wechat-content h2 {
        font-size: 22px;
        border-left: 4px solid #3daeff;
        padding-left: 10px;
      }
      
      /* 引用样式 */
      .wechat-content blockquote {
        background: #f7f7f7;
        border-left: 4px solid #3daeff;
        padding: 10px 15px;
      }
      
      /* 代码块样式 */
      .wechat-content pre {
        background: #f6f8fa;
        padding: 15px;
        border-radius: 4px;
      }
      
      /* 表格样式 */
      .wechat-content table th {
        background: #3daeff;
        color: #fff;
      }
      
      /* 图片样式 */
      .wechat-content img {
        max-width: 100%;
        display: block;
        margin: 15px auto;
      }
    </style>
  `
  
  // 包装内容
  return `${wechatStyles}<div class="wechat-content">${container.innerHTML}</div>`
}
```

### 2. 富文本复制实现

```javascript
case 'wechat':
  const previewHtml = previewRef.current?.innerHTML || ''
  const wechatHtml = convertToWechatFormat(previewHtml)
  
  // 使用 Clipboard API 复制富文本
  if (navigator.clipboard && navigator.clipboard.write) {
    const htmlBlob = new Blob([wechatHtml], { type: 'text/html' })
    const textBlob = new Blob([content], { type: 'text/plain' })
    const clipboardItem = new ClipboardItem({
      'text/html': htmlBlob,
      'text/plain': textBlob
    })
    
    navigator.clipboard.write([clipboardItem]).then(() => {
      showToast('已复制，可直接粘贴到公众号编辑器', 'success')
    })
  } else {
    // 降级方案
    copyWechatHtmlFallback(wechatHtml)
  }
  break
```

### 3. 降级方案实现

```javascript
const copyWechatHtmlFallback = (html) => {
  // 创建临时容器
  const container = document.createElement('div')
  container.innerHTML = html
  container.style.position = 'fixed'
  container.style.left = '-9999px'
  document.body.appendChild(container)
  
  try {
    // 选中内容
    const range = document.createRange()
    range.selectNodeContents(container)
    const selection = window.getSelection()
    selection.removeAllRanges()
    selection.addRange(range)
    
    // 执行复制
    const success = document.execCommand('copy')
    
    // 清理
    selection.removeAllRanges()
    document.body.removeChild(container)
    
    if (success) {
      showToast('已复制，可直接粘贴到公众号编辑器', 'success')
    }
  } catch (err) {
    showToast('复制失败，请手动复制', 'error')
  }
}
```

## 使用方法

### 步骤 1：编写 Markdown 内容

在编辑器中编写你的文章：

```markdown
# 文章标题

这是第一段内容。

## 二级标题

这是第二段内容，包含**粗体**和*斜体*。

### 代码示例

\`\`\`javascript
function hello() {
  console.log('Hello World')
}
\`\`\`

### 引用

> 这是一段引用内容

### 列表

- 列表项 1
- 列表项 2
- 列表项 3

### 表格

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
```

### 步骤 2：导出公众号格式

1. 点击菜单栏"文件" → "导出" → "公众号格式"
2. 看到提示："已复制微信公众号格式到剪贴板"

### 步骤 3：粘贴到公众号编辑器

1. 打开微信公众号后台
2. 进入图文编辑器
3. 在编辑器中按 `Ctrl+V`（Mac: `Cmd+V`）粘贴
4. 样式自动保留，无需手动调整

## 注意事项

### 1. 图片处理

**问题**：本地图片无法在公众号中显示

**解决方案**：
- 使用图床上传图片（如：七牛云、阿里云 OSS）
- 确保图片 URL 可公开访问
- 推荐使用 HTTPS 图片链接

### 2. 外部链接

**微信限制**：公众号文章中的外部链接会被过滤

**建议**：
- 将重要链接放在文末"参考资料"部分
- 使用"阅读原文"功能
- 或转换为二维码图片

### 3. 样式调整

粘贴到公众号后，可能需要微调：
- 检查图片是否正常显示
- 调整段落间距
- 检查表格宽度
- 预览效果

### 4. 特殊元素

**不支持的元素**：
- ❌ Mermaid 流程图（建议导出为图片）
- ❌ 数学公式（建议使用图片）
- ❌ 视频嵌入（使用封面图 + 链接）

**支持的元素**：
- ✅ 标题（H1-H6）
- ✅ 段落
- ✅ 粗体、斜体
- ✅ 引用
- ✅ 代码块
- ✅ 列表（有序、无序）
- ✅ 表格
- ✅ 图片
- ✅ 链接
- ✅ 分隔线

## 故障排除

### 问题 1：复制失败

**症状**：点击导出后提示"复制失败"

**解决方案**：
1. 检查浏览器权限（允许访问剪贴板）
2. 尝试使用 HTTPS 访问
3. 手动选择预览区内容复制

### 问题 2：粘贴后样式丢失

**症状**：粘贴到公众号后没有样式

**解决方案**：
1. 确保使用 `Ctrl+V` 粘贴（不要用右键菜单）
2. 检查浏览器是否支持富文本复制
3. 尝试在公众号编辑器中"粘贴为纯文本"后重新格式化

### 问题 3：图片不显示

**症状**：粘贴后图片显示为占位符

**解决方案**：
1. 确保图片 URL 可公开访问
2. 使用 HTTPS 图片链接
3. 检查图片大小（不超过 2MB）
4. 重新上传图片到公众号素材库

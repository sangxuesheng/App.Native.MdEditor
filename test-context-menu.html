🔧 集成步骤

📝 快速开始

按照以下步骤将右键菜单功能集成到你的 Markdown 编辑器：

  1. 查看 右键菜单功能实现指南.md 了解详细步骤
  2. 查看 EditorContextMenuIntegration.jsx 获取完整代码
  3. 运行集成脚本：bash integrate-context-menu.sh
  4. 手动将代码集成到 App.jsx
  5. 测试所有功能是否正常

# v1.17.0 最终版更新说明

## 🎉 完整功能列表

### 图片上传功能（三种方式）

#### 1. Ctrl+V 粘贴上传 ✅
- 截图后直接粘贴到编辑器
- 自动上传并插入 Markdown 语法
- 支持图片压缩

#### 2. 工具栏上传按钮 ✨
- 点击工具栏的 Upload 图标（上传按钮）
- 支持多选文件
- 批量上传

#### 3. 拖拽上传 ✨
- 从文件管理器拖拽图片到编辑器
- 拖拽时编辑器半透明提示
- 支持批量上传

### 图片管理功能

#### 工具栏图片按钮 🔄 已修改
- **原功能**: 插入图片语法 `![]()`
- **新功能**: 打开图片管理对话框
- **默认标签页**: 图片链接（可以浏览和插入已上传的图片）

**使用流程**:
1. 点击工具栏的图片按钮（Image 图标）
2. 打开图片管理对话框
3. 默认显示"图片链接"标签页
4. 可以：
   - 浏览已上传的图片
   - 点击图片插入到编辑器
   - 切换到"上传图片"标签页上传新图片
   - 切换到"图片库"标签页管理所有图片

## 📋 完整使用指南

### 场景1: 截图后快速插入
**方法**: Ctrl+V 粘贴
1. 截图（Win+Shift+S / Shift+PrtSc）
2. 在编辑器中点击
3. 按 Ctrl+V
4. 自动上传并插入

### 场景2: 浏览并插入已上传的图片
**方法**: 点击工具栏图片按钮
1. 点击工具栏的图片按钮（Image 图标）
2. 自动打开图片管理对话框
3. 默认显示"图片链接"标签页
4. 浏览已上传的图片
5. 点击图片插入到编辑器

### 场景3: 上传本地图片文件
**方法1**: 工具栏上传按钮
1. 点击工具栏的上传按钮（Upload 图标）
2. 选择一个或多个图片文件
3. 自动上传并插入

**方法2**: 拖拽上传
1. 从文件管理器选择图片
2. 拖拽到编辑器区域
3. 松开鼠标
4. 自动上传并插入

### 场景4: 管理已上传的图片
**方法**: 图片管理对话框
1. 点击工具栏的图片按钮
2. 切换到"图片库"标签页
3. 可以：
   - 查看所有图片
   - 预览图片（显示文件名、大小、时间）
   - 批量删除图片

## 🔧 技术改进

### 1. 图片按钮功能优化
**之前**: 插入空的图片语法 `![]()`，需要手动填写 URL
**现在**: 打开图片管理对话框，可以直接选择已上传的图片

**优势**:
- ✅ 更直观，可以看到图片缩略图
- ✅ 不需要记住图片 URL
- ✅ 可以浏览所有已上传的图片
- ✅ 一键插入

### 2. 上传按钮独立
**新增**: Upload 图标按钮
- 专门用于快速上传
- 支持多选
- 不打开对话框，直接上传

### 3. 三种上传方式互补
- **粘贴**: 最快捷，适合截图
- **上传按钮**: 支持多选，适合批量上传
- **拖拽**: 最直观，适合从文件管理器

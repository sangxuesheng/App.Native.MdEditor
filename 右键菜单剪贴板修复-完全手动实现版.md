# 右键菜单剪贴板功能修复 - 完全手动实现版

## 问题分析

### 问题1：Monaco Editor 服务错误
使用 `editor.trigger()` 调用命令时出现：
```
Error: [invokeFunction] unknown service 'productService'
```

### 问题2：execCommand 不工作
`document.execCommand` 在 Monaco Editor 中不能直接工作，因为：
- Monaco Editor 使用自己的选区管理系统
- execCommand 只能操作 DOM 原生的 selection
- 虽然显示"剪切成功"，但实际内容没有被删除

## 最终解决方案

**完全手动实现剪切、复制、粘贴功能**，不依赖任何浏览器命令或 Monaco Editor 内部服务。

## 实现细节

### 1. 剪切操作（Cut）

```javascript
case 'cut':
  if (contextMenu?.selectedText) {
    editor.focus()
    const selection = editor.getSelection()
    const text = editor.getModel().getValueInRange(selection)
    
    // 优先使用 Clipboard API
    if (navigator.clipboard && navigator.clipboard.writeText) {
      navigator.clipboard.writeText(text).then(() => {
        // 删除选中内容
        editor.executeEdits('cut', [{
          range: selection,
          text: ''
        }])
        // 移动光标到删除位置
        editor.setPosition({
          lineNumber: selection.startLineNumber,
          column: selection.startColumn
        })
        showToast('已剪切', 'success')
      }).catch((error) => {
        showToast('剪切失败，请使用 Ctrl+X', 'error')
      })
    } else {
      // 降级方案：使用隐藏 textarea + execCommand
      const textarea = document.createElement('textarea')
      textarea.value = text
      textarea.style.position = 'fixed'
      textarea.style.opacity = '0'
      document.body.appendChild(textarea)
      textarea.select()
      try {
        document.execCommand('copy')
        editor.executeEdits('cut', [{
          range: selection,
          text: ''
        }])
        editor.setPosition({
          lineNumber: selection.startLineNumber,
          column: selection.startColumn
        })
        showToast('已剪切', 'success')
      } catch (err) {
        showToast('剪切失败，请使用 Ctrl+X', 'error')
      } finally {
        document.body.removeChild(textarea)
      }
    }
  }
  break
```

**关键步骤**：
1. 获取 Monaco Editor 的选区
2. 提取选中的文本
3. 写入剪贴板（Clipboard API 或 textarea fallback）
4. 使用 `editor.executeEdits()` 删除选中内容
5. 移动光标到删除位置

### 2. 复制操作（Copy）

```javascript
case 'copy':
  if (contextMenu?.selectedText) {
    editor.focus()
    
    // 优先使用 Clipboard API
    if (navigator.clipboard && navigator.clipboard.writeText) {
      navigator.clipboard.writeText(contextMenu.selectedText).then(() => {
        showToast('已复制', 'success')
      }).catch((error) => {
        showToast('复制失败，请使用 Ctrl+C', 'error')
      })
    } else {
      // 降级方案：使用隐藏 textarea + execCommand
      const textarea = document.createElement('textarea')
      textarea.value = contextMenu.selectedText
      textarea.style.position = 'fixed'
      textarea.style.opacity = '0'
      document.body.appendChild(textarea)
      textarea.select()
      try {
        document.execCommand('copy')
        showToast('已复制', 'success')
      } catch (err) {
        showToast('复制失败，请使用 Ctrl+C', 'error')
      } finally {
        document.body.removeChild(textarea)
      }
    }
  }
  break
```

**关键步骤**：
1. 直接使用 `contextMenu.selectedText`（已经提取好的文本）
2. 写入剪贴板（Clipboard API 或 textarea fallback）
3. 不修改编辑器内容

### 3. 粘贴操作（Paste）

```javascript
case 'paste':
  editor.focus()
  
  // 优先使用 Clipboard API
  if (navigator.clipboard && navigator.clipboard.readText) {
    navigator.clipboard.readText().then(text => {
      const selection = editor.getSelection()
      editor.executeEdits('paste', [{
        range: selection,
        text: text,
        forceMoveMarkers: true
      }])
      // 移动光标到插入内容的末尾
      const lines = text.split('\n')
      const lastLine = lines[lines.length - 1]
      editor.setPosition({
        lineNumber: selection.startLineNumber + lines.length - 1,
        column: lines.length === 1 ? selection.startColumn + lastLine.length : lastLine.length + 1
      })
      showToast('已粘贴', 'success')
    }).catch((error) => {
      showToast('粘贴失败，请使用 Ctrl+V', 'warning')
    })
  } else {
    showToast('粘贴失败，请使用 Ctrl+V', 'warning')
  }
  break
```

**关键步骤**：
1. 从剪贴板读取文本
2. 使用 `editor.executeEdits()` 插入文本
3. 计算插入后的光标位置（考虑多行文本）
4. 移动光标到插入内容的末尾

## 技术要点

### 1. Monaco Editor API 使用

- `editor.getSelection()` - 获取当前选区
- `editor.getModel().getValueInRange(selection)` - 提取选区文本
- `editor.executeEdits(source, edits)` - 执行编辑操作
- `editor.setPosition(position)` - 设置光标位置

### 2. 剪贴板 API 策略

**优先级**：
1. **Clipboard API**（现代浏览器）
   - `navigator.clipboard.writeText()` - 写入
   - `navigator.clipboard.readText()` - 读取
   - 需要 HTTPS 或 localhost
   - 需要用户授权

2. **Textarea + execCommand**（降级方案）
   - 创建隐藏的 textarea
   - 设置内容并选中
   - 使用 `document.execCommand('copy')`
   - 清理 textarea
   - 兼容性最好

### 3. 光标位置计算

**剪切后**：
```javascript
editor.setPosition({
  lineNumber: selection.startLineNumber,
  column: selection.startColumn
})
```

**粘贴后**（单行）：
```javascript
editor.setPosition({
  lineNumber: selection.startLineNumber,
  column: selection.startColumn + text.length
})
```

**粘贴后**（多行）：
```javascript
const lines = text.split('\n')
editor.setPosition({
  lineNumber: selection.startLineNumber + lines.length - 1,
  column: lastLine.length + 1
})
```

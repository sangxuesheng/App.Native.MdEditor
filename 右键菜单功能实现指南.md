# Markdown 编辑器右键菜单功能实现指南

## 功能概述

为 Markdown 编辑器添加了完整的右键上下文菜单功能，支持：

### 图片操作
- ✅ 上传图片
- ✅ 缩放图片（25%, 33%, 50%, 67%, 80%, 100%, 150%, 200%）
- ✅ 转换图片语法（Markdown ↔ HTML）
- ✅ 删除图片文件（仅限本地上传的文件）
- ✅ 拷贝图片标记
- ✅ 将图像另存为

### 编辑操作
- ✅ 剪切、复制、粘贴、删除
- ✅ 工具栏快捷按钮

### 格式化操作
- ✅ 加粗（Ctrl+B）
- ✅ 斜体（Ctrl+I）
- ✅ 行内代码（Ctrl+`）
- ✅ 链接（Ctrl+K）
- ✅ 引用
- ✅ 列表

### 段落操作
- ✅ 标题 1-3
- ✅ 正文

### 插入操作
- ✅ 图片、链接、代码块、表格
- ✅ 无序列表、有序列表、任务列表
- ✅ 分隔线

## 集成步骤

### 步骤 1：导入组件

在 `App.jsx` 顶部添加导入：

```javascript
import EditorContextMenu from './components/EditorContextMenu'
```

### 步骤 2：添加状态

在 App 组件的 useState 声明区域添加：

```javascript
const [contextMenu, setContextMenu] = useState(null)
const [clipboardContent, setClipboardContent] = useState(null)
```

### 步骤 3：添加辅助函数

在 App 组件中添加以下函数（详见 EditorContextMenuIntegration.jsx）：

1. `getSelectedText()` - 获取编辑器选中文本
2. `detectImageAtCursor()` - 检测光标处的图片
3. `detectPreviewImage()` - 检测预览区的图片
4. `handleEditorContextMenu()` - 编辑器右键事件处理
5. `handlePreviewContextMenu()` - 预览区右键事件处理
6. `handleContextMenuAction()` - 菜单操作处理

### 步骤 4：修改 handleEditorMount

在 `handleEditorMount` 函数中添加右键事件监听：

```javascript
const handleEditorMount = (editor) => {
  editorRef.current = editor
  
  // ... 现有代码 ...
  
  // 添加右键菜单事件监听
  const domNode = editor.getDomNode()
  if (domNode) {
    domNode.addEventListener('contextmenu', handleEditorContextMenu)
  }
  
  // ... 现有代码 ...
}
```

### 步骤 5：添加预览区事件监听

添加新的 useEffect：

```javascript
useEffect(() => {
  const previewElement = previewRef.current
  if (previewElement) {
    previewElement.addEventListener('contextmenu', handlePreviewContextMenu)
    return () => {
      previewElement.removeEventListener('contextmenu', handlePreviewContextMenu)
    }
  }
}, [handlePreviewContextMenu])
```

### 步骤 6：添加菜单组件到 JSX

在 App 组件的 return 语句中，在最后的 `</div>` 之前添加：

```javascript
{contextMenu && (
  <EditorContextMenu
    x={contextMenu.x}
    y={contextMenu.y}
    selectedText={contextMenu.selectedText}
    selectedImage={contextMenu.selectedImage}
    theme={editorTheme}
    clipboardHasContent={!!clipboardContent}
    onAction={handleContextMenuAction}
    onClose={() => setContextMenu(null)}
  />
)}
```

## 功能详解

### 自适应定位

菜单会根据鼠标位置和屏幕边界自动调整显示位置：

- **水平方向**：如果右侧空间不足，向左移动
- **垂直方向**：如果底部空间不足，向上移动
- **子菜单**：默认显示在右侧，空间不足时显示在左侧

### 图片检测

系统支持两种图片语法：

1. **Markdown 语法**：`![alt](src "title")`
2. **HTML 语法**：`<img src="..." alt="..." />`

右键点击图片标记时，会自动识别并提供相应操作。

### 图片缩放

通过转换为 HTML 语法并添加 `width` 属性实现：

```html
<img src="image.jpg" alt="图片" width="50%" />
```

### 语法转换

- **Markdown → HTML**：保留 alt 和 src
- **HTML → Markdown**：提取 alt 和 src，转换为 Markdown 格式

### 图片删除

仅对本地上传的图片（路径包含 `/uploads/` 或以 `./` 开头）启用删除功能。

## 快捷键支持

菜单中显示的快捷键：

- `Ctrl+B` - 加粗
- `Ctrl+I` - 斜体
- `Ctrl+\`` - 行内代码
- `Ctrl+K` - 插入链接

## 主题支持

菜单完全支持浅色和暗黑主题，会根据 `editorTheme` 状态自动切换。

CSS 变量：
- `--menu-bg` - 菜单背景色
- `--menu-hover-bg` - 悬停背景色
- `--border-color` - 边框颜色
- `--text-color` - 文字颜色

## 注意事项

1. **图片删除 API**：需要确保后端有 `/api/image/delete` 接口
2. **剪贴板权限**：某些浏览器需要用户授权才能访问剪贴板
3. **性能优化**：使用 `useCallback` 避免不必要的重渲染
4. **事件清理**：确保在组件卸载时移除事件监听器

## ⚠️ 注意事项

### 1. 图片删除功能
需要后端提供 `/api/image/delete` 接口：
```javascript
POST /api/image/delete
Body: { url: "图片URL" }
Response: { ok: true }
```

### 2. 剪贴板权限
某些浏览器需要用户授权才能访问剪贴板。如果遇到权限问题，代码会自动降级使用 Monaco Editor 的内置方法。

### 3. 图片检测
- 支持 Markdown 语法：`![alt](src "title")`
- 支持 HTML 语法：`<img src="..." alt="..." />`
- 光标必须在图片标记内才能检测到

## 🐛 故障排除

### 问题 1：右键菜单不显示
**解决方案**：
1. 检查浏览器控制台是否有错误
2. 确认 EditorContextMenu.jsx 和 EditorContextMenu.css 文件存在
3. 清除浏览器缓存并重新加载

### 问题 2：图片操作不可用
**解决方案**：
1. 确保光标在图片标记内
2. 检查图片语法是否正确
3. 查看控制台日志确认图片检测结果

### 问题 3：剪贴板操作失败
**解决方案**：
1. 检查浏览器剪贴板权限
2. 使用 HTTPS 协议（某些浏览器要求）
3. 代码会自动降级使用 Monaco Editor 方法

### 问题 4：菜单位置不正确
**解决方案**：
1. 检查是否有 CSS 冲突
2. 确认 z-index 设置正确（10000）
3. 查看浏览器窗口大小是否过小

# 图注格式"不显示"时图片居中显示修复

## 问题描述

当在"视图"菜单中选择图注格式为"不显示"时，预览区域的图片布局错误，图片没有居中显示，而是左对齐或布局混乱。

## 问题原因

当图注格式设置为"不显示"时，原代码直接返回 `<img>` 标签，没有包装在 `<figure>` 元素中，也没有添加居中样式，导致图片无法居中显示。

```javascript
// 原代码 - 问题所在
if (caption) {
  return `<figure class="image-figure">
    <img src="${src}" alt="${alt}">
    <figcaption class="image-caption">${caption}</figcaption>
  </figure>`
} else {
  // 没有图注，返回原始 img 标签 - 缺少居中样式
  return `<img src="${src}" alt="${alt}">`
}
```

## 解决方案

### 1. 修改 JavaScript 代码（App.jsx）

为无图注的图片也包装 `<figure>` 元素，并添加特殊的 CSS 类名：

```javascript
// 修改后的代码
if (caption) {
  return `<figure class="image-figure">
    <img src="${src}" alt="${alt}"${title ? ` title="${title}"` : ''}${style ? ` style="${style}"` : ''}>
    <figcaption class="image-caption">${caption}</figcaption>
  </figure>`
} else {
  // 没有图注，包装成 figure 元素但不显示 figcaption，并添加居中样式
  return `<figure class="image-figure image-figure-no-caption">
    <img src="${src}" alt="${alt}"${title ? ` title="${title}"` : ''}${style ? ` style="${style}"` : ''}>
  </figure>`
}
```

**关键改进**：
1. 保留 `style` 属性，支持图片缩放功能
2. 添加 `image-figure-no-caption` 类名用于特殊样式
3. 使用 `<figure>` 元素包装，便于统一管理

### 2. 添加 CSS 样式（App.css）

为无图注的图片添加居中样式：

```css
/* 无图注的图片居中显示 */
.markdown-body .image-figure-no-caption {
  display: flex;
  justify-content: center;
  align-items: center;
  margin: 16px auto;
}

.markdown-body .image-figure-no-caption img {
  display: block;
  margin: 0;
}
```

**样式说明**：
- `display: flex` - 使用 Flexbox 布局
- `justify-content: center` - 水平居中
- `align-items: center` - 垂直居中
- `margin: 16px auto` - 上下间距 16px，左右自动居中

## 技术细节

### Flexbox 居中原理

```css
.image-figure-no-caption {
  display: flex;           /* 启用 Flexbox */
  justify-content: center; /* 主轴（水平）居中 */
  align-items: center;     /* 交叉轴（垂直）居中 */
}
```

### 为什么使用 figure 元素

1. **语义化**：`<figure>` 是 HTML5 语义化标签，专门用于包装图片、图表等内容
2. **统一管理**：有图注和无图注的图片都使用相同的容器结构
3. **样式一致性**：便于统一设置间距、边框等样式
4. **可访问性**：符合 Web 标准，对屏幕阅读器友好

### 保留 style 属性的重要性

```javascript
${style ? ` style="${style}"` : ''}
```

这确保了用户设置的图片缩放（如 `width:50%`）在"不显示图注"模式下仍然生效。

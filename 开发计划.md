## 性能优化详情 (v1.20.11)

### 1. 大文件渲染优化
- 自动检测文件大小（>1MB 为大文件）
- 大文件显示加载提示，避免界面卡顿
- 异步渲染，不阻塞 UI 线程

### 2. 智能自动保存
- 小文件（<100KB）：30秒自动保存
- 中等文件（100KB-1MB）：60秒自动保存
- 大文件（>1MB）：120秒自动保存
- 忽略纯空格变化，减少无效保存

### 3. 防抖渲染
- 使用 500ms debounce 延迟渲染
- 避免频繁输入时的重复渲染
- 提升编辑流畅度

### 4. 性能监控
- 记录每次渲染时间
- 渲染时间 >1秒时给出警告
- 帮助识别性能瓶颈

## 错误提示系统详情 (v1.20.12)

### 1. 统一错误处理
- 创建 `errorHandler.js` 工具模块
- 统一的错误处理接口
- 自动错误类型识别

### 2. 错误类型支持
- 网络错误 (NETWORK)
- 文件未找到 (FILE_NOT_FOUND)
- 权限不足 (PERMISSION_DENIED)
- 文件过大 (FILE_TOO_LARGE)
- 路径无效 (INVALID_PATH)
- 保存失败 (SAVE_FAILED)
- 加载失败 (LOAD_FAILED)
- 渲染失败 (RENDER_FAILED)

### 3. 友好错误信息
- 清晰的错误标题
- 易懂的错误描述
- 针对性的解决建议
- 可选的详细信息

### 4. 详细日志记录
- 结构化的错误日志
- 包含上下文信息
- 时间戳记录
- 便于问题排查

### 5. 应用场景
- 文件加载错误处理
- 文件保存错误处理
- Markdown 渲染错误处理
- 图片上传错误处理

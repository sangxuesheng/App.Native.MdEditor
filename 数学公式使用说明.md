# 数学公式使用说明

## 📐 数学公式语法

Markdown 编辑器支持 KaTeX 数学公式渲染，使用 LaTeX 语法。

---

## 🔤 行内公式

### 语法
使用单个 `$` 包裹公式：

```markdown
这是一个行内公式：$E = mc^2$
```

### 效果
这是一个行内公式：$E = mc^2$

### 更多示例

```markdown
爱因斯坦质能方程：$E = mc^2$

积分公式：$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$

求和公式：$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$

分数：$\frac{a}{b}$

根号：$\sqrt{x}$ 或 $\sqrt[3]{x}$

上下标：$x^2$, $x_i$, $x^{2y}$, $x_{i,j}$
```

---

## 📊 块级公式

### 语法
使用双 `$$` 包裹公式，**必须独立成行**：

```markdown
这是一个块级公式：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### 效果
这是一个块级公式：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

### ⚠️ 重要提示

**块级公式前后必须有空行！**

❌ **错误写法**（没有空行）：
```markdown
这是一个块级公式：$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

✅ **正确写法**（有空行）：
```markdown
这是一个块级公式：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

## 🎯 你的问题解决方案

### 问题 1: 行内公式没有解析

**原因**：可能是格式问题或特殊字符

**解决方案**：
```markdown
❌ 错误：$E = mc^2E=mc2$  （重复了公式）
✅ 正确：$E = mc^2$
```

### 问题 2: 块级公式需要换行

**原因**：块级公式前后没有空行

**解决方案**：
```markdown
❌ 错误：
这是一个行内公式：$E = mc^2$
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

✅ 正确：
这是一个行内公式：$E = mc^2$

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

**关键点**：
1. 块级公式 `$$` 前面必须有空行
2. 块级公式 `$$` 后面必须有空行
3. 公式内容可以多行

## 💡 提示

1. **行内公式**：用于文本中的简短公式，使用单 `$`
2. **块级公式**：用于独立显示的公式，使用双 `$$`，前后必须有空行
3. **错误处理**：如果公式渲染失败，会显示红色错误信息
4. **特殊字符**：某些字符需要转义，如 `\{` `\}` `\_`
5. **对齐**：使用 `align` 环境可以对齐多行公式

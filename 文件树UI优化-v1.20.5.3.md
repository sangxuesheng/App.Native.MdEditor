# 文件树 UI 优化方案 v1.20.5.3

## 更新时间
2025-03-03

## 优化内容

### 1. 隐藏顶部文件路径显示
**原因**: 底部状态栏已经显示了完整的文件路径，顶部显示重复且占用空间

**实现**:
```css
.file-path {
  display: none; /* 隐藏顶部文件路径 */
}
```

**效果**: 
- 顶部工具栏更简洁
- 避免信息重复
- 底部状态栏显示更详细的信息（路径 + 字符数 + 行数 + 保存状态）

---

### 2. 文件树 UI 现代化设计

#### 2.1 整体样式优化
- 增加内边距：12px 16px（更宽松）
- 圆角优化：6px（更柔和）
- 添加悬停动画：向右平移 2px
- 添加阴影效果：激活状态有轻微阴影

#### 2.2 图标和间距
- 图标尺寸：18px × 18px（更大更清晰）
- 图标间距：8px（更宽松）
- 文字大小：13px（保持可读性）

#### 2.3 收藏星标动画
```css
@keyframes starPulse {
  0%, 100% {
    opacity: 1;
    transform: scale(1);
  }
  50% {
    opacity: 0.7;
    transform: scale(0.95);
  }
}
```
- 收藏的文件/文件夹星标会有呼吸动画
- 2秒循环，柔和的缩放效果

---

### 3. 文件夹展开/折叠动画

#### 3.1 展开动画
```css
.tree-node-children {
  overflow: hidden;
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

**特点**:
- 使用 cubic-bezier 缓动函数，更自然
- 300ms 动画时长，流畅不拖沓
- 同时动画高度和透明度

#### 3.2 箭头旋转动画
```css
.tree-node-icon {
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.tree-node-icon.expanded {
  transform: rotate(90deg);
}
```

**效果**: 
- 箭头从右指向下旋转 90 度
- 与内容展开同步
- 视觉反馈清晰

## 技术细节

### CSS 变量系统
```css
.theme-light .file-tree {
  --bg-secondary: #FFFFFF;
  --border-color: #d0d7de;
  --bg-primary: #f6f8fa;
  --text-primary: #24292f;
  --text-secondary: #57606a;
  --hover-bg: #f6f8fa;
  --active-bg: #0969da;
  --active-color: #ffffff;
  --icon-color: #57606a;
}
```

**优势**:
- 统一管理主题颜色
- 易于维护和扩展
- 切换主题无需修改多处代码

### 动画性能优化
- 使用 `transform` 而非 `left/top`（GPU 加速）
- 使用 `cubic-bezier` 缓动函数（更自然）
- 合理的动画时长（300ms）
- 避免同时动画过多属性

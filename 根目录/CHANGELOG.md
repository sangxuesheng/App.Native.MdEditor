# 更新日志 (Changelog)

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.29.30] - 2026-03-27

### 新增
- 预览新增 `infographic` fenced 代码块渲染能力（AntV Infographic），支持 DSL 识别、节点替换与异步渲染
- Infographic 渲染支持图标资源加载器（Iconify），`icon mdi/...` 等资源可在预览中拉取并渲染

### 改进
- Infographic 渲染流程新增事件驱动等待（`rendered/error`）与超时保护，避免 “Infographic is not rendered yet” 时序问题
- Infographic 渲染错误信息可读性增强：支持数组/对象错误展开，输出 `message/reason`，便于定位 DSL 语法问题
- 预览图形容器样式统一（Mermaid/PlantUML/Infographic）：SVG 自适应宽度，修复因固定宽度导致的横向滚动条

### 已知限制
- 当前依赖版本（`@antv/infographic@0.2.16`）与官网 Gallery 模板并非完全一致，部分模板 ID（如 `chart-column-grouped-simple`）在本版本不可用
- `infographic` fenced 代码块当前仅解析 DSL，不直接执行 JS 示例代码（`import/new Infographic(...)`）

## [1.29.29] - 2026-03-27

### 改进
- PlantUML 渲染链路改为前端请求后端 `/api/plantuml/svg` 代理，规避浏览器直连官方服务的 CORS 限制
- PlantUML 代码块解析兼容性增强：放宽 HTML 匹配规则并增加 DOM 二次兜底替换，提升不同渲染结构下的命中率
- PlantUML 预览渲染新增按代码内容缓存，重复图表复用 SVG，减少重复请求与渲染耗时
- 导出一致性增强：HTML、微信公众号与图片导出前统一确保 PlantUML 已渲染，避免“预览有图、导出缺图”

### 修复
- 修复 PlantUML 在部分环境中“请求无报错但未渲染为 SVG”的问题
- 修复 PlantUML 在导出链路中偶发保留源码块而非图形的问题

修复持久化问题

问题：编辑器内容刷新后恢复成默认文本

解决方法：
打开 app/ui/frontend/src/App.jsx
找到第 389 行：} else {
改为：} else if (!savedState?.content) {

完整修改：
将
    } else {
      setContent(`# Markdown 编辑器功能展示

改为
    } else if (!savedState?.content) {
      // 只在没有保存内容时才显示默认文本
      setContent(`# Markdown 编辑器功能展示

保存后重启开发服务器即可。

HEIC 图片格式支持 - 使用 FFmpeg
================================================================================

✅ 实现完成！

功能：自动将上传的 HEIC/HEIF 格式图片转换为 JPEG

技术方案：使用 FFmpeg 在后端进行转换

================================================================================

================================================================================

1. app/server/imageConverter.js (新增)
   - FFmpeg 转换核心模块
   - 支持质量和尺寸参数配置
   - 完善的错误处理

2. app/server/server.js (已更新)
   - 集成 imageConverter 模块
   - 新增 /api/image/converter/status API
   - 修改 /api/image/upload 支持 HEIC 自动转换

3. 安装和测试脚本
   - install-ffmpeg.sh - FFmpeg 安装脚本
   - test-heic-converter.sh - 功能测试脚本
   - test-heic-upload.sh - 上传测试脚本

4. 文档
   - HEIC快速参考.md - 快速参考
   - HEIC_FFmpeg使用指南.md - 完整文档
   - HEIC实现完成总结.md - 实现总结
   - HEIC部署检查清单.md - 部署清单

================================================================================
快速开始
================================================================================

1. 测试功能（FFmpeg 已安装 ✅）
   ./test-heic-converter.sh

2. 启动服务器
   cd app/server && node server.js

3. 测试上传
   ./test-heic-upload.sh /path/to/photo.heic

4. 或在浏览器中直接上传 HEIC 文件

================================================================================
支持的格式
================================================================================

- .heic, .heif - 自动转换为 JPEG
- .jpg, .jpeg, .png, .gif, .webp - 直接保存

================================================================================
API 端点
================================================================================

GET  /api/image/converter/status  - 检查转换工具状态
POST /api/image/upload             - 上传图片（支持 HEIC）

================================================================================
特性
================================================================================

✅ 自动转换 HEIC → JPEG
✅ 批量上传支持

✅ 前端无需修改
✅ 完善的错误处理
✅ 高性能（转换时间 ~200-500ms）

测试 3：检查 CSS 是否生效

使用浏览器开发者工具检查：
  1. 右键点击图片 → 检查元素
  2. 查看 Computed 样式
  3. 检查 width 和 max-width 的值
  4. 如果 max-width: none，说明 CSS 生效
  5. 如果 max-width: 100%，说明 CSS 被覆盖

## 快速开始

### 1. 创建自定义主题

1. 打开导出配置面板（右侧面板）
2. 在"主题"下拉列表中选择"+ 新建自定义"
3. 点击"加载默认模板"按钮获取模板代码
4. 在"自定义 CSS"文本框中编辑样式
5. 点击"保存主题"按钮，输入主题名称
6. 主题会自动出现在下拉列表中

### 2. 使用已保存的主题

1. 在"主题"下拉列表中选择已保存的主题（带有 📦 图标）
2. 主题会自动应用到预览区域
3. 可以随时切换不同的主题

## 支持的 CSS 选择器

所有自定义选择器会被自动转换为标准的 `.markdown-body` 选择器，并添加 `!important` 确保优先级。

### 容器选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `container` | `.markdown-body` | 整个 Markdown 容器，用于设置背景、边距等 |

### 标题选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `h1` | `.markdown-body h1` | 一级标题 |
| `h2` | `.markdown-body h2` | 二级标题 |
| `h3` | `.markdown-body h3` | 三级标题 |
| `h4` | `.markdown-body h4` | 四级标题 |
| `h5` | `.markdown-body h5` | 五级标题 |
| `h6` | `.markdown-body h6` | 六级标题 |

### 文本选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `p` | `.markdown-body p` | 段落 |
| `strong` | `.markdown-body strong, .markdown-body b` | 粗体文本 |
| `link` | `.markdown-body a` | 链接 |

### 列表选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `ul` | `.markdown-body ul` | 无序列表 |
| `ol` | `.markdown-body ol` | 有序列表 |
| `li` | `.markdown-body li` | 列表项 |

### 代码选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `code_pre` | `.markdown-body pre` | 代码块容器 |
| `code` | `.markdown-body pre code` | 代码块内容 |
| `codespan` | `.markdown-body code:not(pre code)` | 行内代码 |

### 其他选择器

| 自定义选择器 | 转换后 | 说明 |
|------------|--------|------|
| `blockquote` | `.markdown-body blockquote` | 引用块 |
| `hr` | `.markdown-body hr` | 分割线 |
| `image` | `.markdown-body img` | 图片 |

## 背景样式设置

### 自动优先级增强

以下 CSS 属性会自动添加 `!important`，确保样式生效：

- `background`
- `background-color`
- `background-image`
- `background-size`
- `background-position`
- `backdrop-filter`
- `-webkit-backdrop-filter`
- `border-radius`
- `box-shadow`
- `padding`
- `border`

### 8种预设背景样式

#### 样式1：基础网格纹理
```css
container {
  background-color: #f8f9fa;
  background-image: 
    linear-gradient(rgba(0, 0, 0, 0.05) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0, 0, 0, 0.05) 1px, transparent 1px);
  background-size: 20px 20px;
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;
}
```
**适用场景**：后台系统、数据展示、通用功能卡片

#### 样式2：渐变光晕背景
```css
container {
  background: linear-gradient(135deg, #f8f9fa 0%, #e9ecef 100%);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1), 
              inset 0 1px 0 rgba(255, 255, 255, 0.8);
  padding: 40px;
}
```
**适用场景**：表单容器、详情页卡片、品牌展示模块

#### 样式3：低饱和度波浪纹理
```css
container {
  background-color: #f8f9fa;
  background-image: 
    repeating-linear-gradient(45deg, rgba(0, 0, 0, 0.03) 0px, rgba(0, 0, 0, 0.03) 2px, 
    transparent 2px, transparent 10px);
  background-size: 28px 28px;
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
  padding: 40px;

```
**适用场景**：文创类页面、轻量级卡片、个人主页模块

#### 样式4：磨砂玻璃纹理
```css

  background-color: rgba(248, 249, 250, 0.85);
  background-image: radial-gradient(rgba(0, 0, 0, 0.02) 1px, transparent 1px);
  background-size: 10px 10px;
  backdrop-filter: blur(8px);
  -webkit-backdrop-filter: blur(8px);
  border-radius: 16px;
```

**适用场景**：叠加在图片/复杂背景上的容器、弹窗、侧边栏
**注意**：需要父级元素有背景才能体现磨砂效果

#### 样式5：极简纯色+微光质感
```css

  background-image: linear-gradient(180deg, rgba(255,255,255,0.8) 0%, rgba(255,255,255,0) 100%);
```

## 主题管理功能

### 保存主题

1. 编辑完 CSS 后，点击"保存主题"按钮
2. 输入主题名称（如：我的蓝色主题）
3. 如果名称已存在，会提示是否覆盖
4. 保存成功后，主题会出现在下拉列表中

### 管理主题

点击"管理主题"按钮，可以：

- **查看**：列出所有已保存的主题
- **编辑**：点击 ✏️ 图标编辑主题
  - 可以修改主题名称
  - 可以修改 CSS 代码
- **删除**：点击 🗑️ 图标删除主题
  - 会弹出确认对话框
  - 如果正在使用该主题，会自动切换到默认主题

### 加载主题

- 在下拉列表中选择主题即可自动加载
- 带有 📦 图标的是已保存的自定义主题
- 选择后会自动应用到预览区域

## CSS 变量

系统提供了一些 CSS 变量，会根据当前设置自动替换：

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `var(--md-primary-color)` | 主题色 | 根据"主题色"设置自动替换 |
| `var(--blockquote-background)` | 引用块背景 | 深色主题：`rgba(255, 255, 255, 0.05)`<br>浅色主题：`rgba(0, 0, 0, 0.03)` |

### 使用示例

```css
h1 {
  color: var(--md-primary-color);
  border-bottom: 2px solid var(--md-primary-color);
}

blockquote {
  background: var(--blockquote-background);
}
```

## 常见问题

### Q1: 为什么我的背景样式没有生效？

**A**: 确保：
1. 使用了 `container` 选择器（不是 `.container`）
2. 刷新了页面或重新选择了主题
3. 检查浏览器控制台是否有 CSS 错误

### Q2: 如何调整网格大小？

**A**: 修改 `background-size` 的值：
```css
background-size: 20px 20px;  /* 20x20 网格 */
background-size: 30px 30px;  /* 30x30 网格（更大） */
background-size: 10px 10px;  /* 10x10 网格（更小） */
```

### Q3: 如何调整网格颜色深浅？

**A**: 修改 `rgba` 的最后一个值（透明度）：
```css
rgba(0, 0, 0, 0.05)  /* 较浅 */
rgba(0, 0, 0, 0.1)   /* 中等 */
rgba(0, 0, 0, 0.2)   /* 较深 */
```

### Q4: 磨砂玻璃效果看不到？

**A**: 磨砂玻璃效果需要：
1. 父级元素有背景（图片或颜色）
2. 使用半透明的 `background-color`
3. 添加 `backdrop-filter: blur(8px)`

### Q5: 如何让主题适配深色模式？

**A**: 系统会自动根据编辑器主题（深色/浅色）调整某些颜色。你也可以使用 CSS 变量：
```css
/* 使用主题色变量，会自动适配 */
h1 {
  color: var(--md-primary-color);
}

/* 使用引用块背景变量，会自动适配深浅色 */
blockquote {
  background: var(--blockquote-background);
}
```

### Q6: 如何导出主题给别人使用？

**A**: 
1. 点击"管理主题"
2. 点击要导出的主题的"编辑"按钮
3. 复制 CSS 代码
4. 分享给其他人，他们可以创建新主题并粘贴代码

### Q7: 可以使用外部字体吗？

**A**: 可以，使用 `@import` 或 `font-family`：
```css
/* 导入 Google 字体 */
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+SC&display=swap');

container {
  font-family: 'Noto Serif SC', serif;
}
```

### Q8: 如何重置为默认样式？

**A**: 
1. 在主题下拉列表中选择"默认"、"经典"、"优雅"或"简洁"等预设主题
2. 或者点击"清空"按钮清空自定义 CSS

## 技术说明

### CSS 转换机制

系统会自动将简化的选择器转换为完整的选择器：

```css
/* 你写的 */
container {
  background: #f8f9fa;
}

/* 实际应用的 */
#preview-area .markdown-body, .markdown-body {
  background: #f8f9fa !important;
}
```

### 优先级保证

所有背景相关的属性都会自动添加 `!important`，确保你的样式不会被覆盖。

### 浏览器兼容性

- 网格背景：所有现代浏览器
- 渐变背景：所有现代浏览器
- 磨砂玻璃（`backdrop-filter`）：
  - Chrome/Edge: ✅ 支持
  - Firefox: ✅ 支持
  - Safari: ✅ 支持（需要 `-webkit-` 前缀）

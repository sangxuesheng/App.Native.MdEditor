# 表格插入功能说明

## 🎉 新增功能

### 可视化表格插入对话框

**位置**:
- 工具栏的表格按钮（Table 图标）
- 菜单栏 → 插入 → 表格

**功能**:
- 可视化选择行数和列数（2-10 行/列）
- 实时预览表格
- 可以直接编辑表头和单元格内容
- 自动生成 Markdown 表格语法

---

## 📋 使用方法

### 方法1: 工具栏按钮
1. 点击工具栏的表格按钮（Table 图标）
2. 打开表格插入对话框
3. 调整行数和列数
4. 编辑表头和单元格内容（可选）
5. 点击"确定"插入表格

### 方法2: 菜单栏
1. 点击菜单栏 → 插入 → 表格
2. 打开表格插入对话框
3. 调整行数和列数
4. 编辑表头和单元格内容（可选）
5. 点击"确定"插入表格

## ✨ 功能特点

### 1. 灵活的行列控制
- **行数**: 2-10 行
- **列数**: 2-10 列
- 使用 [−] [+] 按钮调整
- 实时更新表格预览

### 2. 可编辑的表格内容
- **表头**: 点击输入框编辑表头文字
- **单元格**: 点击输入框编辑单元格内容
- 支持中文输入
- 可以留空

### 3. 自动生成 Markdown
生成标准的 Markdown 表格语法：

```markdown
| 表头1 | 表头2 | 表头3 |
|------|------|------|
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |
```

### 4. 智能插入
- 自动在当前位置插入
- 如果当前行有内容，会在前面添加空行
- 插入后光标移到表格后面

## 🎯 使用场景

### 场景1: 快速创建空表格
1. 点击表格按钮
2. 选择行数和列数
3. 直接点击"确定"
4. 在编辑器中填写内容

### 场景2: 创建带内容的表格
1. 点击表格按钮
2. 选择行数和列数
3. 在对话框中编辑表头和内容
4. 点击"确定"插入

### 场景3: 创建数据表格
1. 点击表格按钮
2. 根据数据调整行列数
3. 填写表头（如：姓名、年龄、地址）
4. 填写数据
5. 点击"确定"

## 🔧 技术实现

### 组件结构
```
TableInsertDialog
├── 头部（标题 + 关闭按钮）
├── 行列控制（计数器）
├── 表格预览（可编辑）
└── 底部按钮（取消 + 确定）
```

### 状态管理
- `rows`: 行数（2-10）
- `cols`: 列数（2-10）
- `headers`: 表头数组
- `cells`: 单元格内容二维数组

### Markdown 生成
1. 生成表头行：`| 表头1 | 表头2 | ... |`
2. 生成分隔行：`|------|------|------|`
3. 生成数据行：`| 内容 | 内容 | ... |`

---

## 💡 使用技巧

### 技巧1: 快速创建
- 默认 3x3 表格，适合大多数场景
- 如果需要调整，使用 [−] [+] 按钮

### 技巧2: 表头优先
- 先填写表头，明确表格结构
- 内容可以在编辑器中慢慢填写

### 技巧3: 留空单元格
- 不需要填写所有单元格
- 空单元格在 Markdown 中显示为空

### 技巧4: 后期编辑
- 插入后可以在编辑器中继续编辑
- Markdown 表格语法简单，易于修改

## 🆚 对比

### 之前的方式
```
点击表格按钮 → 插入固定的表格模板 → 手动修改行列数和内容
```

**缺点**:
- 固定的 3x3 表格
- 需要手动添加/删除行列
- 需要理解 Markdown 语法

### 现在的方式
```
点击表格按钮 → 可视化选择行列数 → 编辑内容 → 一键插入
```

**优点**:
- ✅ 灵活的行列数（2-10）
- ✅ 可视化预览
- ✅ 可以预先填写内容
- ✅ 不需要理解 Markdown 语法
- ✅ 更直观、更高效

# 调试 - 查找图片

## 请在控制台依次执行：

### 1. 检查是否有图片
```javascript
console.log('所有图片:', document.querySelectorAll('.markdown-body img').length)
console.log('带 style 的图片:', document.querySelectorAll('.markdown-body img[style]').length)
console.log('带 width 的图片:', document.querySelectorAll('.markdown-body img[style*="width"]').length)
```

### 2. 查看所有图片的详细信息
```javascript
const allImages = document.querySelectorAll('.markdown-body img')
console.log('=== 所有图片 ===')
allImages.forEach((img, i) => {
  console.log(`图片 ${i+1}:`, {
    src: img.src.substring(0, 50) + '...',
    alt: img.alt,
    style: img.getAttribute('style'),
    hasWidth: img.getAttribute('style')?.includes('width'),
    offsetWidth: img.offsetWidth,
    computedMaxWidth: window.getComputedStyle(img).maxWidth
  })
})
```

### 3. 如果没有图片，检查 markdown-body 是否存在
```javascript
const markdownBody = document.querySelector('.markdown-body')
console.log('markdown-body 存在:', !!markdownBody)
if (markdownBody) {
  console.log('markdown-body 内容长度:', markdownBody.innerHTML.length)
  console.log('markdown-body 内容预览:', markdownBody.innerHTML.substring(0, 200))
}
```

### 4. 检查预览区
```javascript
const preview = document.querySelector('.preview-pane')
console.log('preview-pane 存在:', !!preview)
if (preview) {
  console.log('预览区所有图片:', preview.querySelectorAll('img').length)
}
```

---

## 可能的情况：

1. **还没有输入测试代码** - 需要在编辑器中输入图片代码
2. **图片还没渲染** - 需要等待 Markdown 渲染完成
3. **选择器不对** - 图片可能不在 `.markdown-body` 中

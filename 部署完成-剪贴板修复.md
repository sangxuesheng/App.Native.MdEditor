## 修复内容

### 核心修改
1. **完全手动实现剪贴板操作**
   - 不依赖 Monaco Editor 内部服务
   - 不依赖 `document.execCommand`
   - 直接使用 Monaco Editor API 操作文本

2. **剪切操作**
   - 使用 `navigator.clipboard.writeText()` 写入剪贴板
   - 使用 `editor.executeEdits()` 删除选中内容
   - 正确设置光标位置

3. **复制操作**
   - 使用 `navigator.clipboard.writeText()` 写入剪贴板
   - 不修改编辑器内容

4. **粘贴操作**
   - 使用 `navigator.clipboard.readText()` 读取剪贴板
   - 使用 `editor.executeEdits()` 插入文本
   - 正确计算并设置光标位置（支持多行）

5. **降级方案**
   - Clipboard API 失败时，使用隐藏 textarea + execCommand
   - 确保在 HTTP 和 HTTPS 环境都能工作
